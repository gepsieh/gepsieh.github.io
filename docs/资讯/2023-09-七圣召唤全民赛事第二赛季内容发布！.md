七圣召唤全民赛事第二赛季内容发布！

=================

![](https://upload-bbs.miyoushe.com/upload/2023/09/27/384454482/2dba7342fc1c30c934abe9f4a0ee4368_9209740339313822085.png)

亲爱的旅行者们，七圣召唤全民赛事第二赛季即将开启！
**赛季时间：2023年9月-2024年4月**
一起来看看本赛季的赛事内容吧！
_(烟绯-得意)
// 穿越层峦千壑，一睹召唤之巅的风采——
第二赛季开启三大晋级赛道「全民积分赛」、「平台资格赛」及「最终入围赛」，最终将角逐出128名顶尖牌手奔赴「影幻杯」公开赛舞台，登上最强召唤王的决斗赛场！
其中，「全民积分赛」征程将持续4个版本，排行榜机制也迎来升级。更激烈的对局从此刻开启！
_(夜兰-打盹)
// 循迹胜利曙光，赢下召唤之巅的奖赏——
第二赛季总奖金投入约为185万元。
本赛季将发布全新赛季主题牌背，包含1张「影幻杯荣誉牌背」，及2张「全民赛限定牌背」，牌背获取规则将在原神赛事官网公布。此外，原石及赛事奖金也将持续激励牌手们的对决。用一场关键的胜负来赢下奖赏吧！

![](https://upload-bbs.miyoushe.com/upload/2023/09/27/384454482/09fd9149aa89081b55d645fafcb3a4b4_5797895823993965674.jpg)

**新的征程已经开始，期待与旅行者们在召唤之巅相会！**
