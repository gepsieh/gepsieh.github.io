七圣召唤全民赛事第二赛季牌背公布！

=================

亲爱的旅行者们，久等了！
七圣召唤全民赛事「第二赛季牌背」即将正式上线~

埋藏于千岩深壑下的宝藏，是给予冒险家勇气的奖赏
正如牌手们追求的极境，也要通过不断试炼才能抵达……

★ 公开赛荣誉牌背「层岩巨渊：冠冕」
黄金代表最为辉煌的财富，无论冒险家、矿工还是牌手，都将这色彩视为事业的冠冕……
// 牌背获取条件：参与七圣召唤公开赛事，并获得优异的战绩。

★ 全民赛限定牌背「层岩巨渊：坚铁」
坚铁之色的矿体，是农具与兵器的底色。
层岩巨渊蕴藏的诸多矿产，仍是令人敬畏的巨大财富……
// 牌背获取条件：参与4.1及4.2版本开展的七圣召唤全民赛事，并获得可观的战绩。

★ 全民赛限定牌背「层岩巨渊：靛晶」
靛青之色的矿体，是冒险之路向远方延展的信号。
层岩巨渊蕴藏的诸多矿产，仍是令人敬畏的巨大财富……
// 牌背获取条件：参与4.3及4.4版本开展的七圣召唤全民赛事，并获得可观的战绩。
