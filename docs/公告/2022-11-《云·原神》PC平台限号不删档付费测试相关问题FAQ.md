《云·原神》PC平台限号不删档付费测试相关问题FAQ

亲爱的旅行者：
《云·原神》PC平台限号不删档付费测试已于2022/11/07 11:00 正式开启。在这里我们会从测试相关、体验相关、计费相关三个方面解答旅行者的问题。

▍测试相关问题
Q1：通过《云·原神》PC平台玩《原神》体验会有什么不同？
A1：旅行者通过下载《云·原神》PC端安装包，登录米哈游通行证即可在PC设备上体验《原神》游戏，不需要再下载完整《原神》包体；且您在《云·原神》PC平台上的游戏进度与其他平台互通。

Q2：测试会影响我的《原神》进度吗？
A2：旅行者通过《云·原神》PC端安装包体验到的《原神》游戏进度与其他平台互通，本次测试过程中的《原神》游戏进度将正常保存。

Q3：如何获取测试资格？
A3：旅行者可以点击下方链接，完成问卷并提交，即可参与《云·原神》PC平台限号不删档付费测试资格招募报名。我们将抽取部分成功提交问卷的旅行者，分批发放测试资格。

*招募截止时间为2022/11/11 18:00

Q4：本次测试时间维持多久？
A4：本次测试已于2022/11/7 11:00正式开启，持续时间请以官方通知为准。

Q5：《云·原神》PC平台可以使用哪些服务器账号进行登录？
A5：目前，《云·原神》PC平台仅支持天空岛服务器账号登录。

Q6：《云·原神》PC平台的《防沉迷说明》和《公平运营声明》是否与其他客户端保持一致？
A6：是的，保持一致。例如，某法定节假日20时至21时期间，实名信息为未成年人（未满18周岁）的旅行者A，已通过PC平台体验了0.5小时《原神》游戏。根据新颁布的《关于进一步严格管理 切实防止未成年人沉迷网络游戏的通知》，当该旅行者通过《云·原神》登录《原神》时，剩余的累计在线时长不能超过0.5小时，超时将无法登录《云·原神》PC平台。

▍体验相关问题
Q1：如何进行《云·原神》PC平台的设置（如画质、网络状态显示）？
A1：旅行者可以点击屏幕左侧悬浮球，进行画质设置和开关网络状态显示。

Q2：如何设置《云·原神》PC平台的全屏或窗口模式？
A2：旅行者可以点击主界面右上角第3个设置按钮，通过【游戏设置】选择全屏或窗口模式（支持选择窗口大小）。

Q3：《云·原神》PC平台是否支持手柄功能？
A3：目前《云·原神》PC平台暂不支持手柄功能，相关信息请旅行者们留意后续公告。

Q4：《云·原神》PC平台是否支持按键自定义设置？
A4：目前《云·原神》PC平台暂不支持按键自定义设置，相关信息请旅行者们留意后续公告。

Q5：如何切换账号？
A5：点击主界面右上角的【登出】按钮即可进行账号切换。

Q6：《云·原神》PC平台的设备系统要求是怎样的？
A6：目前《云·原神》PC平台支持的最低配置如下：
操作系统: Windows 7 SP1 64-bit 或 Windows 8.1 64-bit 或 Windows 10 64-bit
处理器: Intel Core i3 或同等处理器
内存:4 GB RAM
显卡NVIDIA GeForce GTX 750
DirectX 版本: 9

Q7：为什么我在体验游戏时会觉得很卡？
A7：《云·原神》PC平台的游戏体验对旅行者的网络有较高要求，建议旅行者在网络状况良好时体验游戏，或者尝试在游戏内的悬浮球中手动将画质切换为标清等较低档位。

Q8：如何退出《云·原神》PC平台？
A8：旅行者打开悬浮球点击左上角的【返回主界面】，或者直接关闭《云·原神》PC平台的窗口，即可退出游戏，退出游戏后将不再进行计费。

Q9：当我在体验《云·原神》PC平台的过程中遇到问题应该在哪里进行反馈？
A9：测试期间，旅行者遇到任何问题，都可以在[《云·原神》游戏问题集中反馈](https://bbs.mihoyo.com/ys/article/30916248)贴内进行反馈；或通过以下两种渠道联系客服，我们将尽快在第一时间为旅行者解决：
- 通过《云·原神》PC平台主界面【问题反馈】或【悬浮球】-【问题反馈】进行提交。
- 如遇到《原神》游戏内的问题，推荐通过游戏内【派蒙菜单】-【反馈】-【左下角"+"号】-【问题反馈】进行提交，该通道含有账号坐标、报错等重要信息，更有助于程序定位问题。

▍计费相关问题
Q1：如何获取免费时长？
A1：若旅行者首次使用任意平台（PC平台、安卓及iOS平台）登录《云·原神》，将会获得5小时的免费时长。此外，每日首次登录游戏时，可领取15分钟的免费时长（每日凌晨04:00刷新）。免费时长累积上限为600分钟，达到600分钟上限后则无法继续累积。
*《云·原神》PC平台账号与安卓、iOS平台账号互通，同一账号的免费时长、畅玩卡及米云币余额为各平台共享。
★免费时长的发放可能会存在几分钟延迟，请旅行者们耐心等待哦~

Q2：什么是畅玩卡？
A2：旅行者开通畅玩卡后，在畅玩卡生效期间无需消耗米云币与免费时长即可畅玩游戏。若当前账户内无生效中的畅玩卡，则购买当天至次日0时享受免费畅玩，次日0时起开始计算畅玩卡生效时间。畅玩卡的价格为60元/30日。
★只有当畅玩卡剩余生效时间＜180天时续费，总生效时间才会延长；当畅玩卡剩余生效时间≥180天时，旅行者将无法继续购买畅玩卡。
但如因特殊情况导致重复购买，总生效时间将无法进行累加，系统将在付费成功后按300米云币/1元的比例向旅行者返还等价的米云币。（例如：旅行者A当前畅玩卡剩余生效时间为180天，该旅行者因为特殊情况支付60元购买了30日畅玩卡，则该旅行者的畅玩卡总生效时间仍为180天，系统会向该旅行者返还18000米云币）

Q3：《云·原神》PC平台测试期间的计费体系是怎样的？
A3：旅行者在《云·原神》PC平台启动游戏时即开始计费。计费规则为10米云币/分钟。若账户内有剩余免费时长或生效中的畅玩卡，则按照【畅玩卡】-【免费时长】-【米云币】的顺序进行扣费。

Q4：如何购买米云币？
A4：旅行者可通过点击信息界面或悬浮球中的【充值】来购买米云币，米云币有以下5个充值档位：

Q5：我通过任务管理器关闭《云·原神》PC平台时米云币余额为1000，为什么重新登录变成了980？
A5：若旅行者通过任务管理器结束《云·原神》PC平台任务的方式退出，2分钟后才会彻底断开连接并停止扣费哦~

Q6：我在《云·原神》PC平台可以正常进行游戏内付费（如凝取结晶、开启空月祝福等）吗？
A6：可以正常进行游戏内付费，会和旅行者平常在游戏内的购买流程一样，且旅行者在《云·原神》PC平台的游戏内付费记录与其他平台互通。

Q7：我在《云·原神》APP内购买的米云币可以在《云·原神》PC平台使用吗？
A7：可以使用。《云·原神》PC平台账号与安卓、iOS平台账号互通，同一账号的免费时长、畅玩卡及米云币余额为各平台共享。

Q8：我在《原神》游戏内购买当期纪行所对应的珍珠之歌（128元档），免费获得价值60元的《云·原神》「30日畅玩卡」可以在PC平台使用吗？
A8：可以使用，请旅行者记得在购买珍珠之歌后的7*24小时内使用同一账号登录任意《云·原神》平台激活「30日畅玩卡」，逾期后「30日畅玩卡」将失效且无法补发。
