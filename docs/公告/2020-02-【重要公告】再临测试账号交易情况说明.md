【重要公告】再临测试账号交易情况说明

==================

![](https://upload-bbs.miyoushe.com/upload/2020/09/23/74757616/88daf15a31d86efd760f9907713592ba_1963592238551917049.png)

近期，旅行者们在各个平台发现了游戏安装包出售与测试账号的共享/交易的情况，并且积极地进行了反馈，感谢各位旅行者对我们工作的支持。我们正在根据旅行者们提供的相关信息进行调查与处理。
 
目前，我们已根据反馈的信息，取消了部分相关账号的测试资格，**并将视情况取消这些出售测试资格的旅行者后续所有测试资格，对其米游社账号进行永久封禁。**
 
请广大旅行者们保护好个人财产，不要轻信任何游戏安装包出售与测试账号共享/交易的信息，切勿在其他平台购买游戏安装包或测试账号。
同时，也请已获得测试资格的旅行者保护好个人信息，遵守测试要求，请勿共享/出售自己的测试账号。
 
客服娘再次郑重提醒各位旅行者：
**我们严禁一切账号的共享/交易行为，一经发现将取消该账号的测试资格。如旅行者们发现存在账号交易行为，请保留证据并私信原神玩家版主/迷路的史莱姆酱/飞翔的丘丘人/原神米游姬（旅行者们任选其一进行私信即可）进行举报。由于私信功能暂不支持图片发送，相关图片可发送给官方群群管理进行举报。谢谢！**
 
暂时没有得到测试资格的旅行者也不要心急，接下来还会开放更多获取测试资格的活动，期待大家的参与，说不定下一个幸运儿就是你哦ღ( ´･ᴗ･` )
