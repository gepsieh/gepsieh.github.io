PC版下载安装常见问题FAQ

==============

亲爱的旅行者：
在PC版预下载开启后，项目组收到了旅行者提出的关于下载、玩法等许多问题，我们在此一一为大家进行解答。旅行者有任何问题，都可以通过游戏内反馈页面，或是在米游社APP内私信1号客服娘或2号客服娘进行咨询与反馈~

**▌机型适配**
**Q1：PC推荐配置有哪些**
A1：操作系统：Windows 7 SP1 64-bit 或 Windows 8.1 64-bit 或 Windows 10 64-bit（暂不支持 Windows 10预览版dev）
处理器：Intel Core i7 或同等处理器
内存：16 GB RAM
显卡：NVIDIA GeForce GTX 1060 6GB 及更高配置
DirectX 版本：11
存储空间：请预留 **30 GB **可用空间
 
**PC最低配置**
操作系统：Windows 7 SP1 64-bit 或 Windows 8.1 64-bit 或 Windows 10 64-bit（暂不支持 Windows 10预览版dev）
处理器：Intel Core i5 或同等处理器
内存：8 GB RAM
显卡：NVIDIA® GeForce® GT 1030 及更高配置
DirectX 版本：11
存储空间：请预留 **30 GB** 可用空间
 
**▌常见PC版安装问题**
Q1：打开启动器或启动游戏时，提示：此文件的版本与正在运行的Windows版本不兼容。
A1：请确认Windows系统是否满足游戏支持的适配要求（见文首推荐配置列表），若设备性能较低，可能会出现无法安装、无法启动、黑屏、卡顿等现象。
PS：Windows7操作系统需达到Windows 7 Service Pack 1版本（补丁包SP1）才可支持游戏。

![](https://upload-bbs.miyoushe.com/upload/2020/09/13/75276550/abfc79762e43adb54a9205ecfd659399_446804804016940444.png)

 
Q2：打开启动器后，启动器主界面展示范围异常。（如启动器页面展示不完整）
A2：右键原神桌面图标属性后选择【兼容性】-【更改高DPI设置】勾选【替代高DPI缩放行为】，下面选【系统】，应用后重启启动器，可恢复正常显示范围。

Q3：下载过程中提示：检查网络连接。
A3：请点击重新下载，若一直报错，请检查网络和*设置。
 
Q4：点击开始游戏，游戏程序无响应，无法启动游戏。
A4：请确认是否有安装杀毒软件，并确认原神安装路径中的相关文件是否有被拦截，若有被拦截，请尝试取消拦截，或信任文件，或退出杀毒软件后再重新打开游戏。
 
Q5：启动游戏后，未出现原神logo，一直处于白屏状态。
A5：请查看启动器安装目录中是否存在“DXSETUP”文件夹，若有，请双击该文件夹内的“DXSETUP.exe”尝试手动安装后再重启游戏；若没有，请下载安装DirectX 11。 
 
Q6：启动游戏时提示：资源文件下载失败或网络问题，请重试 Code：-9908。
A6：可尝试多次重启游戏，若仍无法解决，请旅行者卸载启动器并删除游戏安装包，重新安装原神。
 
Q7：更新游戏时提示：磁盘空间不足。
A7：请确认游戏安装磁盘剩余空间大小，若不足30G，建议清理剩余空间后重新打开启动器更新游戏。

Q8：我先前测试的安装包未卸载，我可以直接安装《原神》技术性开放测试PC版吗？
A8：请在安装《原神》技术性开放测试PC版时，先卸载先前测试的版本。

Q9：运行原神时遇到如下两种提示：
1、应用程序无法正常启动(0xc000007b)。请单击“确定”关闭应用程序。
2、由于找不到MSVCP140.dll，无法继续执行代码。重新安装程序可能会解决此问题。
A9：请在启动器安装目录下，找到“vcredist2013_x64.exe”双击进行安装，
并去下方微软官网下载安装最新的VC_redist.x64.exe ，再重新运行游戏。
[https://support.microsoft.com/zh-cn/help/2977003/the-latest-supported-visual-c-downloads](https://support.microsoft.com/zh-cn/help/2977003/the-latest-supported-visual-c-downloads)
 

![](https://upload-bbs.miyoushe.com/upload/2020/09/13/75276550/7386c338deee9c7f093a3613fac0659d_2127210497758482285.png)

Q10：下载安装过程中遇到杀毒软件拦截，或出现以下弹窗提示：

![](https://upload-bbs.miyoushe.com/upload/2020/09/13/75276539/bc965ce5fe901db0ca5c53b81b158ee8_7954893685928023575.png)

A10：下载安装打开游戏过程中，若出现杀毒软件拦截提示，请点击允许或信任操作，或退出杀毒软件。（若遇到360拦截，请重启电脑后再打开游戏） 

Q11：点击启动器无反应。
A11：请按照A9的提示下载更新VC_redist.x64.exe 。
若完成上述操作后仍无法解决，请查看当前设备是否同时具备独立显卡和集成显卡，若是，请禁用集成显卡后重新运行启动器。若否，请于米游社APP内，私信账号1号客服娘或2号客服娘进行咨询。

**▌常见游戏问题**
Q1：能否跨服联机？
A1：9月15日开放的服务器为：天空岛服
目前只有在同一个服务器的旅行者能够互相进行联机。在同一服务器的情况下，达到一定的冒险等阶且完成相应任务后，旅行者可申请加入彼此的游戏世界，参与联机活动。使用 PS4 平台进行游戏的旅行者，也可在官方服务器中，加入其他平台旅行者好友的游戏世界，一同体验丰富的联机活动。
 
Q2： 我参加了「启程测试」，要如何获得充值返还呢？「启程测试」的充值返还什么时候可以拿到？9月15日还是9月28日？
A2：旅行者使用「启程测试」中进行过充值的同一账号登录游戏，开启邮件功能后，即可收到返利邮件。
 
通过官方渠道和TapTap获得「启程测试」资格并进行充值的旅行者可在9月15日进入游戏后，获得在「启程测试」的充值返还。
通过B站获得「启程测试」资格并进行充值的旅行者可在9月28日进入游戏后，获得在「启程测试」的充值返还。

Q3：游戏时画面卡顿怎么办？
A3：旅行者首先可检查所使用的设备是否符合目前的设备要求（见文首推荐配置列表），其次您也可以在派蒙菜单中点击设置，适当降低画质相关选项以提高整体流畅程度。
如果以上方法无法解决旅行者的问题，可在游戏内反馈-联系客服页面下方点击【加号+】按钮，联系人工客服进行反馈。
 
Q4：无法下载/更新游戏怎么办？
A4：首先请旅行者确认自己所使用的设备有足够的存储空间以安装《原神》，其次由于下载安装文件较大，我们推荐您使用相对稳定的网络环境进行下载，并确保有足够的网络流量，如进行以上操作后，若仍未解决问题，可以在米游社APP内，私信账号1号客服娘或2号客服娘进行咨询。
 
Q5：有未成年防沉迷吗？限制是什么样的？
A5：有，具体规则如下：
【在线时长限制】
1）法定节假日：累计在线时长不能超过3小时，超时将无法继续游戏；
2）其他时间：累计在线时长不能超过1.5小时，超时将无法继续游戏。
3）每日22:00~次日08:00期间无法登录游戏。
【充值限制】
1）0岁~7岁（含7岁）的旅行者无法充值；
2）8岁~16岁（含16岁）的玩家，单笔充值不得超过50元人民币，每月累计充值不超过200元人民币；
3）17~未满18岁的玩家，单笔充值不得超过100元人民币，每月累计充值不超过400元人民币。
 
 
Q6：什么时候可以玩到移动端平台？
A6：《原神》全球同步公测（iOS、Android、PC、PS4）将于9月28日正式开启。
官网预约地址：[https://ys.mihoyo.com/pre-register/?utm_source=wbpz#/](https://ys.mihoyo.com/pre-register/?utm_source=wbpz#/)
 
Q7：我是TapTap下载的游戏，我可以和在官网下载游戏的朋友一起联机么？
A7：可以。
 
Q8：我参加了PC测试，手机端开服后，可以登录PC账号，继承游戏进度吗？
A8：可以，15号所开启的服务器为天空岛服，同一账号在该服内，安卓平台、iOS平台与PC平台，游戏数据互通（继承进度）。由于PSN账号限制，旅行者无法使用PSN账号，通过其他平台登录游戏。
 
Q9：我是PS4玩家，想和其他朋友进行联机可以么？
A9：可以，旅行者需要和您的朋友选择同一服务器，都达到冒险等阶16级，并且完成相关游戏任务后，才能联机。由于平台政策要求，PSN账号地区为中国大陆的旅行者将进入官方服务器（天空岛服）。

Q10：《原神》PC版支持MacOS系统吗？
A10：《原神》PC版暂不支持MacOS系统。

Q11：《原神》PC版支持Xbox/PS4手柄吗？接入手柄，更改设置无法操作怎么办？
A11：
1. 目前《原神》PC版原神支持PS4手柄和Xbox one/精英手柄，请旅行者确认手柄是否属于支持型号。
2. 请旅行者到系统【控制面板】-【设备与打印机】自查是否有虚拟手柄输入造成冲突。
 
Q12：9月15日在PC注册登录后，同一账号9月28日在其他移动端上可以充值吗？
A12：9月15日在PC注册登录后，同一账号9月28日在其他移动端上也可以充值。
 
Q13：PC版技术性开放测试限号删档吗？
A13：PC版技术性开放测试不限号、不删档。

Q14：本次PC测试支持虚拟机吗？
A14：PC版技术性开放测试不支持虚拟机。

Q15：进入游戏无故闪退，是因为什么？
A15：引起【闪退】这个表现的原因有很多，开发组已经在排查可能存在的问题，建议旅行者将显卡切换为独立显卡后，再进行游戏。
如何切换成独立显卡：
1. NVIDIA显卡
右键桌面的空白位置>「Nvidia控制面板」>3D设置 > 管理3D设置>程序设置>选择要自定义的程序，下拉框选择原神
>第二个下拉菜单里，选择该程序偏好的显卡处理器。

2. AMD独立显卡
右键桌面的空白位置>并选择「Radeon设置」>首选项 > 更多设置 > 电源 > 可切换显示卡全局设置 （或者：显示卡属性 > 性能 > amd radeon dual graphics  > 启用 amd radeon dual graphics ）
>程序列表中选择原神。
（如果游戏不在列表里，点击「添加程序」按钮，并从游戏安装路径中，选择游戏的这个执行文件*.exe）
>在「显卡设置」栏，将「高性能」(或“最大化性能”）分配给游戏。
