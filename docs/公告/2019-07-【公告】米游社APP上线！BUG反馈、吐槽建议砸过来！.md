【公告】米游社APP上线！BUG反馈、吐槽建议砸过来！

===========================

![](https://upload-bbs.miyoushe.com/upload/2020/09/23/74757616/88daf15a31d86efd760f9907713592ba_5265384842645702723.png)

旅行者们，好久不见，你们的霸霸又来了（想死我？）
硬霸出场，必有大事。不卖关子，这次要介绍的，就是大家期（gu）待（le）已（hen）久（jiu）的米游社APP！
米游社APP会给旅行者带来哪些船新体验？让我一一道来：
 
1.初次访问米游社APP，旅行者可自由选择关注的游戏。
最少需要选一个，懂我的意思吧（疯狂暗示）。

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/8b93b21fbdcbebac1b309dd05c6e7b1c_1586242451685497506.png)

 
如果不小心选多或者漏选，也不要紧。可以从首页设置游戏展示游戏频道，随时调整关注的游戏。（认准了，红框里的就是设置入口）
 

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/64bfbd102395bae6c1e10c00ff31b30b_4810418463091012896.png)

 
频道设置页面就是下面介个样子，旅行者们可以拖动选择首页展示的频道。

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/6e272bdc3ee46f400130599d2f0d0a4b_7839238395468787768.png)

 
设置完成后，大家就可以从首页浏览关注的游戏内容，及时发现新活动和讨论热点。

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/7536b37560a66af1ffba4fb8231abae1_948998947002776115.png)

 
旅行者们可能已经注意到了首页搜索栏旁边的签到按钮。目前米游社不同游戏是分开签到的，**每个游戏的经验值也是独立的**。多修的同学不要忘记分别打卡。

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/0496de3b434fefeba23dbc067b89af46_4015038967424976378.png)

在APP签到之后，就能很方便地查看社区任务。快来看看今天的社区经验有没有拿满！

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/86040884efea29c8a9f33d55e3693625_2960847843558057014.png)

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/49c3f94fbe9f80fcbed27826dac29252_3084085748234088312.png)

 
2.在米游社获得的关注、赞、回复和社区通知都会集中到APP的消息页面，接收消息再也不用切换游戏！

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/1b7ceee69d696a3f9953551faf62d487_3013422623338969226.png)

 
3.想知道关注的大佬又发了哪些帖？戳“论坛-关注”随时get大佬动态！

![](https://upload-bbs.miyoushe.com/upload/2019/07/29/72465771/adf5eb5c087b1d59b22d9c98a158217d_634583308143560677.png)

 
**APP我们先行开放了安卓下载~IOS下载也将在近期开放，敬请期待。**
 
米游社体验的改进离不开旅行者们的支持。**如果旅行者们对APP有任何想吐槽的地方欢迎在本帖留言**。各位旅行者的建议是我莫大的动力→关注是双倍的动力。我真的不是来骗关注的，看我的眼神！最后，感谢旅行者们对米游社的反馈，吐槽和建议都拿小本本记下了，新功能在做了在做了，肯定不会咕！
