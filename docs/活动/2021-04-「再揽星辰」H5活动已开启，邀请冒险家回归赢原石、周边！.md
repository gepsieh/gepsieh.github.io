「再揽星辰」H5活动已开启，邀请冒险家回归赢原石、周边！

============================

冒险家协会发布「再揽星辰」邀请函，邀请活跃冒险家携手回归冒险家一同冒险，探索原石、实物周边等多种好礼。
[>>点击参与活动<<](https://webstatic.mihoyo.com/ys/event/e20210428invite/index.html?bbs_auth_required=true&bbs_presentation_style=fullscreen&bbs_landscape=true&utm_source=bbs&utm_medium=mys&utm_campaign=arti)
 
**「再揽星辰」H5活动说明**
**【活动时间】**
活动时间：2021/04/28 ~ 2021/05/17 23:59
抽奖时间：2021/04/28 ~ 2021/05/20 23:59
* 2021年5月20日23:59后将无法进行抽奖，请及时领取探索礼券并完成抽奖。
 
**【参与条件】**
**=回归冒险家=**
截至1.5版本更新停服维护结束时，需同时满足以下条件：
1- 冒险等阶≥10级
2- 距离最后一次登录游戏时间 ≥14天
3- 距离上一次激活游戏内「星之归还」活动＞45天（不包含激活当天）
 
*注：「星之归还」——回归冒险家可参与的游戏内活动。参与活动可获得原石*500、脆弱树脂*3、四星武器「单手剑·试作斩岩」等奖励。
 
**=活跃冒险家=**
冒险等阶≥10级的非回归冒险家

方式一：

方式二：米游社APP-原神版区首页-再揽星辰

**【玩法介绍】**
活动期间内，活跃冒险家、回归冒险家均可通过完成活动任务领取探索能量值，获取当前能量值所对应的探索礼券。使用礼券可参与礼券抽奖，抽取原石、实物周边、大英雄的经验等多种好礼！
 
**=邀请码填写=**
回归冒险家进入活动页面后，可在「受邀启程」页面填写有效邀请码，获得探索礼券。

![](https://upload-bbs.miyoushe.com/upload/2021/04/28/75276539/56c88053901ccc494f56a5cc277c1b51_1309034613453277708.png)

**=探索能量值=**
2021年5月17日23:59前，可在「获取礼券」页面完成对应活动任务，领取探索能量值。
 
**=探索礼券=**
1- 探索能量值达到节点后可以领取对应的探索礼券。2021年5月20日23:59前，可在「获取礼券」页面获取当前探索能量值对应的探索礼券。可领取的探索礼券数量存在上限，领取完毕后探索能量值可能存在一定溢出。
2- 每消耗1张探索礼券可以进行1次探索抽奖。2021年5月20日23:59前，可在「礼券抽奖」页面使用探索礼券参与抽奖，除了原石、大英雄的经验等虚拟奖励外，还有较小的概率抽到实物周边。
3- 抽奖实时出奖，可在「礼券抽奖」页面点击「奖励一览」按钮，查看有机会获取的奖品详情。

1- 实物周边共有以下5种：

2- 所有实物周边均为该种类中的任意一款随机发货。
3- 实物周边存在一定的制作周期，预计将在七月上旬开始陆续发货。
4- 若出现因冒险家错误填写地址信息而无法寄达的情况，不再补发。
 
**【周边寄送地址填写】**
1- 成功探索到实物周边的冒险家可点击探索中奖弹窗的「填写地址」按钮，填写个人联系方式。
2- 未在探索中奖弹窗填写地址信息的冒险家可点击导航栏右上角宝箱按钮，打开「我的奖励」弹窗，查看中奖信息及填写地址。
3- 目前地址信息仅支持填写一次，确认提交之后将无法再次自主修改。若因不可控因素出现误填情况，请在2021年5月31日23:59前联系客服沟通确认后进行修改。
4- 2021年5月20日23:59后，将无法再填写地址信息。若超时未填写地址则默认放弃奖励，请冒险家们及时填写。
