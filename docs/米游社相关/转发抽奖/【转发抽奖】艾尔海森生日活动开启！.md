2. 收到中奖的系统通知后，旅行者可以点击该通知，直接前往中奖详情页，在详情页中点击【点击提交收货地址】按钮，填写收货地址即可。
3. 若获奖旅行者发现活动中周边等实体奖品的相关售后问题（包括但不限于误发、漏发、商品质量问题等），可通过米游社客服反馈（详细操作：米游社App内点击【我的】-选择【联系客服】按钮进入反馈页面-选择《原神》前往客服页-进行问题反馈），在查明证实后视具体情况处理，售后问题的受理期限为相关活动开奖之日起9个月。
