![image](https://upload-bbs.miyoushe.com/upload/2024/02/23/75276545/94648bc214c266508bc949bfb0191add_6377134237340464315.png)活动期间内，**转发本帖到自己的动态**，即视为参与成功。

![image](https://upload-bbs.miyoushe.com/upload/2024/02/23/75276545/e6fc972d96ab8e076815698245dd5f13_1943573363170267999.png)1. 到达开奖时间后，系统将会结算中奖结果，并自动转发原帖@前三名中奖的旅行者（超出3名时，后续中奖的旅行者将会被省略）。
2. 请旅行者关注系统通知，周边中奖信息会出现在里面，如果中奖旅行者没有在指定时间前完成地址填写，则视为放弃奖项；如果旅行者获得了周边奖励，请按照以下步骤填写地址：[【戳我看填写地址教程】](https://bbs.mihoyo.com/ys/article/48940)。
3.若获奖旅行者发现活动中周边等实体奖品的相关售后问题（包括但不限于误发、漏发、商品质量问题等），可通过米游社客服反馈（详细操作：米游社App内点击【我的】-选择【联系客服】按钮进入反馈页面-选择《原神》前往客服页-进行问题反馈），在查明证实后视具体情况处理，售后问题的受理期限为相关活动开奖之日起9个月。
