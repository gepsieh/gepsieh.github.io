Olah，旅行者们好呀~又到了一年海灯节，璃月港换上了节日的装扮，似乎还有很多人从远方赶来，看起来十分热闹。本丘的小伙伴送来了好多灯笼，说是等晚上一起赏灯玩，看来今天本丘的烤肉仓库要遭殃啦——不过能和小伙伴一起过节玩耍就是最开心的，对不起了仓库~
![image](https://upload-bbs.miyoushe.com/upload/2023/02/03/75276545/3089ba12aca5c7689d70fd44454071bb_3307343216175708000.png)灯笼包裹里还有一沓小纸条，有几张写着奇怪的文字…“相识两月成知己。（字）”什么意思？——原来是谜语呀！纸条背面写着答案，不过酱酱说不给旅行者们看~坏酱酱！
时间不早了，本丘祝各位旅行者在新的一年里平安快乐，事业学业节节高哦！(づ￣ 3￣)づ

**活动期间，转发本活动帖到自己的动态，还有机会获得本丘给各位旅行者准备的节日礼物哦~**

![image](https://upload-bbs.miyoushe.com/upload/2023/02/03/75276545/66e4f8a84a5e3e2ef1b92ae703993c03_1811971025081797985.png)2023年2月5日-2023年2月8日23:59
开奖时间：活动结束后系统会结算中奖结果，并自动转发原帖@前三名中奖的旅行者

![image](https://upload-bbs.miyoushe.com/upload/2023/02/03/75276545/8559f2f6da26b0fc65481ace1d5fb995_8233952857928957169.png)活动期间内，转发本帖到自己的动态，即视为参与成功。
 *旅行者们可以点击主帖底部的【转发】按钮进行动态转发；
![image](https://upload-bbs.miyoushe.com/upload/2023/02/03/75276545/c4344b6351ebb2e7dfb2498a3154920a_7793320973030508382.png)*目前动态功能仅支持米游社手机App，使用米游社网页端的旅行者们可以进入米游社App参与本活动~

![image](https://upload-bbs.miyoushe.com/upload/2023/02/03/75276545/552024ae185f3ed7df16e9e447dd8997_9064580711413877863.png)原神主题系列随机角色护身符璃月款*1

本丘将从成功参与活动的旅行者中，随机抽取30名送出「原神主题系列随机角色护身符璃月款*1」的奖励。

![image](https://upload-bbs.miyoushe.com/upload/2023/02/03/75276545/2ab6df6932af9afc59b6ddef96995b83_8029163078513796005.png)1. 到达开奖时间后，系统将会结算中奖结果，并自动转发原帖@前三名中奖的旅行者（超出3名时，后续中奖的旅行者将会被省略）。
2. 收到中奖的系统通知后，旅行者可以点击该通知，直接前往中奖详情页，在详情页中点击【点击提交收货地址】按钮，填写收货地址即可。
