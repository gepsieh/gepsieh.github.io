3月10日00：00-23：59期间，旅行者可以进入「留影叙佳期」页面，为申鹤庆祝生日并收集相应的「画片」。
[>>点击进入展示页<<](https://webstatic.mihoyo.com/ys/event/e20220303-birthday/index.html?game_biz=hk4e_cn&bbs_presentation_style=fullscreen&bbs_auth_required=true&bbs_landscape=true&activity_id=20220301153521&utm_source=bbs&utm_medium=mys&utm_campaign=arti)
（3月11日0：00之后，旅行者仍然可从米游社APP-旅行工具中进入展示页，但将无法收集画片，请注意时间）

![image](https://upload-bbs.miyoushe.com/upload/2023/03/09/76387920/d8ec60736f6e54cfce75fecabc07f53d_114229286965723221.png)2023年3月10日-2023年3月12日18:00
开奖时间：活动结束后系统会结算中奖结果，并自动转发原帖@前三名中奖的旅行者

![image](https://upload-bbs.miyoushe.com/upload/2023/03/09/76387920/417a820d8542bfe5db2c86dfe47852c1_7921315254691711648.png)活动期间内，**转发本帖到自己的动态**，即视为参与成功。
 *旅行者们可以点击主帖底部的**【转发】**按钮进行动态转发；
![image](https://upload-bbs.miyoushe.com/upload/2023/03/09/76387920/660bbde11accd608643d9c762fe8da77_2303293075961928769.png)*目前动态功能仅支持米游社手机App，使用米游社网页端的旅行者们可以进入米游社App参与本活动~

**P.S 今天是申鹤的生日，不要忘记为她送上祝福哦~**

![image](https://upload-bbs.miyoushe.com/upload/2023/03/09/76387920/79d1b021ddd48f70ef35d6da4bb6ebad_1436749842917837559.png)申鹤角色立牌*10
申鹤角色徽章*30

![image](https://upload-bbs.miyoushe.com/upload/2023/03/09/76387920/ddcdde202e6e0ce690e77d1daeac9438_6246992738073029902.png)1. 到达开奖时间后，系统将会结算中奖结果，并自动转发原帖@前三名中奖的旅行者（超出3名时，后续中奖的旅行者将会被省略）。
2. 收到中奖的系统通知后，旅行者可以点击该通知，直接前往中奖详情页，在详情页中点击【点击提交收货地址】按钮，填写收货地址即可。
