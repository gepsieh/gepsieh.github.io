开奖时间：活动结束后系统会结算中奖结果，并自动转发原帖@前三名中奖的旅行者

![image](https://upload-bbs.miyoushe.com/upload/2022/05/20/76387920/8559f2f6da26b0fc65481ace1d5fb995_6686868142107984706.png)活动期间内，**转发本帖到自己的动态**，即视为参与成功。
 *旅行者们可以点击主帖底部的**【转发】**按钮进行动态转发；
![image](https://upload-bbs.miyoushe.com/upload/2022/05/20/76387920/c4344b6351ebb2e7dfb2498a3154920a_7281725975304286504.png)*目前动态功能仅支持米游社手机App，使用米游社网页端的旅行者们可以进入米游社App参与本活动~

![image](https://upload-bbs.miyoushe.com/upload/2022/05/20/76387920/2ab6df6932af9afc59b6ddef96995b83_7846146288886809096.png)1. 到达开奖时间后，系统将会结算中奖结果，并自动转发原帖@前三名中奖的旅行者（超出3名时，后续中奖的旅行者将会被省略）。
2. 收到中奖的系统通知后，旅行者可以点击该通知，直接前往中奖详情页，在详情页中点击【点击提交收货地址】按钮，填写收货地址即可。
