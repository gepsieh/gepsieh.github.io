10月19日00：00-23：59期间，旅行者可以进入「留影叙佳期」页面，为早柚庆祝生日并收集相应的「画片」。
[**>>点击进入展示页<<**](https://webstatic.mihoyo.com/ys/event/e20220303-birthday/index.html?game_biz=hk4e_cn&bbs_presentation_style=fullscreen&bbs_auth_required=true&bbs_landscape=true&activity_id=20220301153521&utm_source=bbs&utm_medium=mys&utm_campaign=arti)
（10月20日0：00之后，旅行者仍然可从米游社APP-旅行工具中进入展示页，但将无法收集画片，请注意时间）

![image](https://upload-bbs.miyoushe.com/upload/2022/10/18/76387920/dbd19b141212f2eea24daca4ae3c300b_5825350002383611066.png)2022年10月19日-2022年10月21日18:00
开奖时间：活动结束后系统会结算中奖结果，并自动转发原帖@前三名中奖的旅行者

![image](https://upload-bbs.miyoushe.com/upload/2022/10/18/76387920/bd0b8ce2259f73c1ffa7bcada0198f02_2549961789261019084.png)活动期间内，**转发本帖到自己的动态**，即视为参与成功。
 *旅行者们可以点击主帖底部的**【转发】**按钮进行动态转发；
![image](https://upload-bbs.miyoushe.com/upload/2022/10/18/76387920/c463e0a85de000f0243a91aec35900f2_5755734440928187636.png)*目前动态功能仅支持米游社手机App，使用米游社网页端的旅行者们可以进入米游社App参与本活动~

**P.S 今天是早柚的生日，不要忘记为她送上祝福哦~**

早柚角色徽章*30

2. 收到中奖的系统通知后，旅行者可以点击该通知，直接前往中奖详情页，在详情页中点击【点击提交收货地址】按钮，填写收货地址即可。
