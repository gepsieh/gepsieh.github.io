**〓更新内容抢先看〓**
**▌全新内容**
**元素共鸣**
新增了系统：元素共鸣。
若当前队伍中的四名角色，满足某些元素属性组合时，可以激活对应的特殊效果。
可在队伍配置界面，查看当前生效的元素共鸣效果，以及所有元素共鸣效果一览。
 
**原素烘炉**
新增了活动：原素烘炉。可通过蒙德城内炼金学者NPC「蒂玛乌斯」参与活动，在大世界中进行玩法挑战。
 
**魔晶萃取**
新增了魔晶萃取功能。
大世界中新增了一种矿物：魔晶矿。可在铁匠NPC处，消耗原粹树脂对魔晶矿石进行锻造，获得可观数量的精锻用魔矿（武器经验素材）。
 
**矿点查询**
旅行者与铁匠NPC对话，可获得当前大世界中矿物位置的刷新信息。
 
▌活动祈愿「闪焰的驻足」
1. 祈愿时间
2020/07/03 12:00 ~ 2020/07/16 11:59
 
2. 祈愿介绍
活动期间内，限定五星角色「逃跑的太阳·可莉(火)」的祈愿获取概率将大幅提升！
活动期间内，四星角色「未授勋之花·诺艾尔(岩)」「少年春衫薄·行秋(水)」「万民百味·香菱(火)」的祈愿获取概率将大幅提升！
活动结束后，「逃跑的太阳·可莉(火)」将不会出现在其他祈愿中，而「未授勋之花·诺艾尔(岩)」「少年春衫薄·行秋(水)」「万民百味·香菱(火)」仍可以在常驻祈愿中获取。
 
**▌调整&优化**
调整了冒险等阶25级突破任务秘境中，最终奖励宝箱的位置，避免玩家因误操作退出秘境错过奖励。
增加了无相元素的重生时间提示。    
增加了以下NPC出售的商品限购数量，防止误操作导致浪费大量摩拉：    
     舒茨、布洛克、快刀陈、苏二娘、纪芳、朱老板、菲尔戈黛特    
降低了璃月地区每日委托「临危受命」中遗迹猎者的生命值。
 
**▌问题修复**
※任务    
修复了璃月魔神任务「往生」中，北国银行NPC叶卡捷琳娜有可能无法正常对话的问题。   
修复了传说任务：小兔之章，阅读飞行指南后可能黑屏的问题。    
修复了传说任务：幼狮之章中，无法正常与NPC玛格丽特、优律进行任务对话的问题。    
修复了任务「宝藏归离」中，在水位下降动画播放过程中退出游戏，会造成水位恢复的问题。    
修复了传说任务：幼狮之章中，在向玛格丽特询问暗夜英雄的信息前重新登录游戏，会导致玛格丽特离开的问题。    
修复了突发事件「圆滚滚的大团骚乱」中，驱逐尘埃后掉线，可能无法正常完成事件的问题。 
    
※秘境    
修复了秘境「暴风般狂啸之龙」中，特瓦林的生命值显示位置不正确的问题。    
修复了深境螺旋第3层第2间的怪物阵容预览中，冰深渊法师错误地显示为水深渊法师的问题。    
修复了PC端在进入秘境后，大幅占用系统盘空间的问题。    
    
※角色&武器    
修复了多人游戏中，其他玩家的角色死亡会导致弓类角色的准星显示错的问题。    
修复了角色突破等级超过4后，角色一览界面该角色突破等级仍显示为4的问题。    
修复了使用迪卢克制作食物「轻策农家菜」时，固有天赋无法正确增加额外熟练度的问题。
修复了部分角色在攀爬或跳跃时施放元素爆发，有概率触发该角色已装备的圣遗物四件套效果的问题。    
修复了菲谢尔通过元素战技召唤出的奥兹，无法攻击风魔龙特瓦林的问题。    
修复了弓手在瞄准状态下可以无视地表移动的问题。    
修复了祈愿中，武器「以理服人」错误地显示为突破3阶后外观的问题。    
    
※其他    
修复了一部分宝箱无法被正常解锁打开的问题。 
修复了食物「厚云朵松饼」没有使用冷却时间的问题。    
修复了果酒湖中的风元素方碑有概率不产生风场，导致无法获取上方风神瞳的问题。    
修复了无相元素在重生阶段中，偶尔会重复刷新创生物的问题。    
修复了燃烧的草地无法被触发超载反应的问题。    
修复了达达乌帕谷的擂台挑战开启后，传送离开再返回后屏障不消失的问题。    
修复了进行天赋升级时，图标显示不正确的问题。    
修复了在烹饪未结束时关闭界面后，每次关闭烹饪界面都会重复播放完成特效的问题。    
修复了多人游戏中，聊天记录超过2个屏幕长度后，快速滑动聊天记录会导致记录不显示的问题。    
修复了若与北风的王狼战斗战败，纪行相关条目仍能完成的问题。    
修复了从武器强化界面退出到武器界面时，角色手上的武器消失的问题。    
修复了大量的场景穿模、卡死等问题。    
修复了部分NPC位置、行为异常的问题。    
修复了部分音效无法正常播放的问题。    
修复了矿石偶尔无法被击碎的问题。    
修复了部分移动设备、蓝牙设备游玩时出现的杂音、爆音、单声道等问题。
