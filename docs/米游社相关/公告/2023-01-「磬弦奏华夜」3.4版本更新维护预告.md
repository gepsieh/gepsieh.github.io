「磬弦奏华夜」3.4版本更新维护预告

==================

![](https://upload-bbs.miyoushe.com/upload/2023/01/13/75276539/9a0b13afcee10467b17301fbb046123a_6068829558119381127.jpg)

亲爱的旅行者：
制作组将于近期进行版本更新维护，维护期间无法登录游戏。请旅行者留意游戏时间。
维护完成后游戏将更新为全新版本，建议旅行者在WIFI 环境下升级。

**〓更新时间〓**
2023/01/18 06:00开始，预计5个小时完成。

**〓客户端更新方式〓**
PC：关闭当前游戏后，重新打开启动器，点击更新；
iOS：进入App Store，点击更新；
安卓：进入游戏后，按照游戏内弹窗提示完成更新。
PS5™或PS4™：打开系统界面-选择原神-按下手柄OPTIONS键-选择确认升级。
若遇到任何安装问题，请联系客服，我们将为旅行者解决问题。

**〓补偿内容〓**
停服更新补偿：原石*300（停服维护每小时补偿60原石）

**〓补偿范围〓**
停服更新补偿：2023/01/18 06:00前冒险等阶达到5级及以上的旅行者。
领取时间截至3.4版本结束前。
制作组将在维护完成后5小时内，以游戏内邮件的形式，向旅行者发放相应补偿。邮件有效期为30天，请及时领取附件。
更多更新内容，包括问题修复、问题修复补偿等详见将于2023/01/18 07:00发布的版本更新说明。

〓调整及优化〓
● 系统
1. 「还圣奥迹」中，切换圣匣后不会清空已选中的圣遗物；
2. 「还圣奥迹」中，选中已强化过的圣遗物时增加了二次确认弹窗；
3. 优化了「还圣奥迹」中圣遗物的筛选条件；
4. 「还圣奥迹」中，圣遗物可滑动批量选择；

● 音频
1. 优化了「七圣召唤」部分元素反应效果的音效；
2. 调整了角色天气相关语音和闲聊语音同时触发时的播放逻辑；

● 角色
1. 优化了角色「兰生幽谷·夜兰(水)」的手部模型外观表现；
2. 调整了角色「梦园藏金·多莉(雷)」的角色立绘的腿部表现；

● 七圣召唤
1. 新增使用手柄游玩「七圣召唤」时，在光标悬停状态下，PC端和PS4™端长按交叉键，PS5™长按圆圈键可以结束回合（以DualSense™ 或DUALSHOCK®4无线控制器为例）的功能；
2. 新增使用手柄游玩「七圣召唤」时的L1和R1按键提示（以DualSense™ 或DUALSHOCK®4无线控制器为例）；
3. 调整了「七圣召唤」中角色牌「宵宫」的元素爆发所需充能数，元素骰子数量和造成的伤害：所需充能数由2调整为3，所需骰子数量由3个火元素骰子调整为4个，“造成3点火元素伤害……”调整为“造成4点火元素伤害……”；
4. 调整了「七圣召唤」中角色牌「魔偶剑鬼」元素战技「孤风刀势」和「霜驰影突」造成的伤害：这两个元素战技将不再造成伤害，仅分别召唤剑影·孤风和剑影·霜驰；
5. 调整了「七圣召唤」中事件牌「兽肉薄荷卷」的效果：现在其效果最多能触发3次；
6. 调整了「七圣召唤」中阵营出战状态「激化领域」的可用次数：由3次调整为2次；
7. 调整了「七圣召唤」中天赋牌「飞叶迴斜」所需元素骰子数量：所需骰子数量由3个草元素骰子调整为为4个；
8. 优化了「七圣召唤」中，部分角色牌牌面的外观表现；
9. 优化了使用移动端游玩「七圣召唤」时，卡牌附着护盾的特效表现；
10. 优化了「七圣召唤」中，牌局内新角色牌加入时的动画效果表现；

● 其它
1. 调整了成就“训练有素的考古学家”的描述；
2. 调整了成就“越过沙暴与蜃气楼·其一”所需解锁的传送锚点数量。原先所需解锁锚点总数为26，调整后为27（若该成就已完成，则成就完成状态保持不变）；
3. 调整了元素共鸣效果的生效规则：当队伍中存在4名角色，或队伍因试用角色存在4名及以上角色时，元素共鸣效果即可正常生效，具体元素共鸣效果由队伍中第1至第4名角色的对应元素决定（调整前，若存在试用角色则元素共鸣效果不会生效）；
4. 取消了BOSS「正机之神」战斗二阶段过场动画的跳过动画二次确认弹窗，现在点击跳过会直接跳过动画，无需再确认；
5. 调整了部分突发事件触发的高度判定范围。
