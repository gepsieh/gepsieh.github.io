「赤土之王与三朝圣者」3.1版本更新维护预告

======================

![](https://upload-bbs.miyoushe.com/upload/2022/09/23/75276557/99de9bbd275c8b16811ebc72b6c76a69_7144287810958035174.png)

亲爱的旅行者：
制作组将于近期进行版本更新维护，维护期间无法登录游戏。请旅行者留意游戏时间。
维护完成后游戏将更新为全新版本，建议旅行者在WIFI 环境下升级。

〓更新时间〓
2022/09/28 06:00开始，预计5个小时完成。

〓客户端更新方式〓
PC：关闭当前游戏后，重新打开启动器，点击更新；
iOS：进入App Store，点击更新；
安卓：进入游戏后，按照游戏内弹窗提示完成更新。
PS5™或PS4™：打开系统界面-选择原神-按下手柄OPTIONS键-选择确认升级。
若遇到任何安装问题，请联系客服，我们将为旅行者解决问题。

〓补偿内容〓
停服更新补偿：原石*300（停服维护每小时补偿60原石）

〓补偿范围〓
停服更新补偿：2022/09/28 06:00前冒险等阶达到5级及以上的旅行者。
领取时间截至3.1版本结束前。
制作组将在维护完成后5小时内，以游戏内邮件的形式，向旅行者发放相应补偿。邮件有效期为30天，请及时领取附件。
更多更新内容，包括问题修复、问题修复补偿等详见将于2022/09/28 07:00发布的版本更新说明。

〓调整及优化〓
● 敌人
1. 降低了BOSS「急冻树」从瘫痪中恢复后，施放的霜霰攻击技能连续命中角色时造成的伤害；
2. 调整了敌人技能伤害的计算方式（部分敌人的特定技能伤害会出现微量的下降，其他敌人的技能伤害不变）；
3. 优化了「镀金旅团」部分扬沙特效的效果；

● 系统
1. 深境螺旋在每一间切换上下半时，将清理上半敌人、角色所产生的创生物（残留的元素晶球、元素微粒以及结晶反应产生的晶片不会被清理）；
2. 调整优化了背包内部分道具的排序规则；
3. 减少了尘歌壶内部分摆设的摆放限制；
4. 版本更新后，凝取结晶将进行首充双倍重置。届时凝取结晶界面的所有档位都会重置为首充双倍状态。
• 重置后，每个档位都可以享受首充双倍返利。
• 重置之前未进行过首充的档位，其双倍返利次数不会保留，即不会在重置后触发2次首充双倍返利。
5. 角色在受击和部分技能状态下，将不再能使用派蒙菜单的拍照功能（留影机不受影响）；
6. 优化了在祈愿获得草元素角色时，草元素图标的动态效果表现；
7. 调整了「毗波耶」的刷新间隔，调整后其间隔与水晶矿一致；

● 音频
1. 优化了部分解密物件的音效表现；
2. 优化了小道具「老旧的诗琴」的音效表现
3. 优化了击倒敌人和暴击时的音效表现；
4. 优化了部分任务的日语、韩语语音，优化了部分NPC的韩语语音；
5. 优化了部分道具相关的英语语音；
6. 新增了部分角色在沙漠环境下触发相关语音的功能；

● 其它
1. 略微调整了NPC哈尔瓦与NPC沙班达尔的位置；
2. 调整了「真识波」的特效表现；
3. 调整了沙漠地形下陷痕迹的消失速度；
4. 角色邀约界面的标题从“传说任务/邀约事件”改为“邀约事件”；
5. 调整了「须弥」地区个别位置的矿石种类：将部分魔晶矿调整为水晶矿；
6. 调整了部分场景的碰撞体积；
7. 调整了敌人「遗迹龙兽」在瘫痪和未吸收能量状态下弱点的特效表现。
