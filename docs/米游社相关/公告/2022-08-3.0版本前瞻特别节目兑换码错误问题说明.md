3.0版本前瞻特别节目兑换码错误问题说明

====================

![](https://upload-bbs.miyoushe.com/upload/2022/08/13/75276557/380130ee07fd3c0978176ea1bc64fdf9_3128017160553466480.png)

亲爱的旅行者，
在刚结束的3.0版本前瞻特别节目中，我们公布的第三个兑换码有误。
非常抱歉给各位旅行者带来了不便。
经过我们的调整，原错误的兑换码也可以进行兑换。即MAFVJAPQSEWN和MAFVJAPOSEWN均可进行兑换，两个兑换码的对应奖励一致，但仅可选择其中一个进行兑换。

以下为本次前瞻特别节目的三个兑换码，旅行者可以前往游戏内兑换领取相应奖励：
ZBXCKAN9SFFE
GSWV2BNRAFX2
MAFVJAPQSEWN

同时本次兑换码兑换时间将延长到2022/08/14 23:59，请旅行者及时兑换。
