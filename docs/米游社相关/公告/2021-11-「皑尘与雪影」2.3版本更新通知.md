「皑尘与雪影」2.3版本更新通知

================

![](https://upload-bbs.miyoushe.com/upload/2021/11/22/75276557/7e3bdee895b989571a8757ff9920941e_1914448980211866765.jpg)

亲爱的旅行者：
为了给旅行者更好的游戏体验，制作组将于近期进行版本更新停服维护，维护完成后游戏将更新为全新版本。
旅行者重新下载客户端后即可继续进行体验。此次更新资源较大，建议旅行者在WiFi环境下进行版本升级。
维护期间旅行者无法登录游戏，请旅行者留意游戏时间，以避免造成不必要的损失。因此带来的不便，还请旅行者理解> < 

〓更新时间〓
2021/11/24 06:00开始，预计5个小时完成。

〓客户端更新方式〓
PC：关闭当前游戏后，重新打开启动器，点击更新；
iOS：进入App Store，点击更新；
安卓：进入游戏后，按照游戏内弹窗提示完成更新。
PlayStation®4及PlayStation®5：打开系统界面-选择原神-按下手柄OPTIONS键-选择 确认升级。
若遇到任何安装问题，请联系客服，我们将为旅行者解决问题。

〓补偿内容〓
停服更新补偿：原石*300
（停服维护每小时补偿60原石。若提前开服，补偿将不会改变。）

〓补偿范围〓
停服更新补偿：2021/11/24 06:00前冒险等阶达到5级及以上的旅行者。
领取时间截至2.3版本结束前。 
制作组将在维护完成后5小时内，以游戏内邮件的形式，向旅行者发放相应补偿。邮件有效期为30天，请及时领取附件。

八、其他新增内容
● 玩法
1.新增食谱：「鳗肉茶泡饭」、「五宝腌菜」、「绯樱虾仙贝」、「常胜传说」、「强者之道」；
2.「天地万象」和「心跳的记忆」中新增部分成就；
3.新增名片：「纪行·白英」、「五郎·悠犬」、「一斗·鬼颜」；
4.新增摆设：闲趣机关-「冲刺节拍」、「萦绊之刻」等；
5.尘歌壶入驻系统新增同伴：派蒙；
6.优化了套装回收功能：可以一键从家园中回收指定套装中缺失的摆设（该摆设不可处于其他套装中）；
7.调整了背包-圣遗物页签、角色圣遗物替换界面的默认圣遗物排序规则，调整后圣遗物默认排序规则为：品质＞等级＞套装＞位置＞词条数量；
8.圣遗物获取界面新增锁定功能；
9.点击背包内部分角色培养素材来源的“星尘兑换”后，将直接跳转星尘兑换页面；
10.使用「深秘圣物匣」后，可在获取界面查看圣遗物属性；
11.新增了部分加载页面提示；
12.在联机状态下，有队友退出时，会按照进入联机之前的单人队伍配置进行队伍补充角色；
13.深境螺旋
     第十一层地脉异常调整为：
     • 角色造成的岩元素伤害提升60%。
     • 角色造成的物理伤害提升60%。
     更新了深境螺旋第十一层~十二层的怪物阵容。
     从2.3版本更新完成后的第一次深境期数更新开始，三期渊月祝福分别如下：
     第一期：
     萃炼之月
     获得结晶反应形成的晶片后，会获得一层萃炼：当前场上角色造成的伤害提升8%，持续10秒，至多叠加3层。处于3层萃炼效果下的角色获得结晶反应形成的晶片后，将释放冲击波，对附近的敌人造成真实伤害。每3秒至多通过这种方式释放一次冲击波。
      第二期：
      灵石之月
      当前场上角色身边存在岩元素创造物时，重击造成的伤害提升30%，消耗的体力降低50%。
      第三期：
      纯色之月
      角色通过普通攻击、重击、下落攻击、元素战技或元素爆发对敌人造成元素伤害后，队伍中所有角色获得10%对应的元素伤害加成，并每3秒将进一步获得10%该元素伤害加成，直到通过该渊月祝福的效果获得100%元素伤害加成为止。当角色通过上述方式对敌人造成了其他元素的伤害后，此前获得的元素伤害加成将被清除。

● 角色
1.角色「智明无邪·烟绯(火)」、「忍里之貉·早柚(风)」、「黑羽鸣镝·九条裟罗(雷)」各新增了一套待机动作；

● 其他
1.角色和敌人的技能伤害、元素反应伤害、机关物件等新增单次伤害上限，最大不会超过9999999；
2.新增对雷蛇设备幻彩功能的支持。若旅行者游戏时使用的为支持幻彩功能的雷蛇设备，则需先升级雷云控制面板升级至3.6.515以上版本，并启动雷云控制面板以开启设备的幻彩功能。

〓调整&优化〓
● 角色
1.优化了角色「渡来介者·托马(火)」部分待机动作中团子消失的时间点；

● 敌人
1.降低了水飘浮灵水球技能的跟踪能力；
2.降低了水飘浮灵水球技能、岩飘浮灵岩标技能的打断能力；
3.优化了飘浮灵在石化、冻结状态下被击败时的爆炸过程；
4.减小了兽境猎犬的被击退时的动作位移距离；
5.优化了遗迹机兵在冻结状态下的材质表现效果；

● 音频
1.调整和优化了部分角色和部分任务中的日语、韩语、英语语音；
2.调整和优化了小道具「镜花之琴」在演奏时的音色；

● 其它
1.「稻妻」地区，逢岳之野区域一处「若雷结界」中，增加了「雷樱枝条」；
