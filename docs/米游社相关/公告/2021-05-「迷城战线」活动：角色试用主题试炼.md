「迷城战线」活动：角色试用主题试炼

=================

![](https://upload-bbs.miyoushe.com/upload/2021/05/15/75276539/8095b3625eb4f23a79068e46e5753c99_8125125194673279649.jpg)

活动期间，旅行者可以使用指定的试用角色编队并挑战6个不同主题的试炼，完成试炼任务可获得原石、大英雄的经验、天赋培养材料等奖励。
 
**〓活动时间〓**
2021/05/21 10:00 ~ 2021/05/31 03:59
 
**〓参与条件〓**
冒险等阶≥20级
 
**〓活动说明〓**
● 「迷城战线」共有6种不同主题的试炼，从活动第一天起，每2天解锁2种新试炼。
● 在「迷城战线」的试炼中，需要在指定时间内寻找并激活所有的「古代符文」，进行「最终挑战」。
● 每个试炼拥有不同的地脉异常，敌人和陷阱机关的分布也有所不同。
● 试炼中只能使用给定的试用角色，也无法产生元素共鸣。
● 若旅行者拥有试用角色，在试炼中，该试用角色将继承旅行者实际拥有角色的命之座层数。若旅行者暂未拥有试用角色，命之座将默认为0层。
