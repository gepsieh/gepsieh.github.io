亲爱的旅行者：
以下是4.0版本——「仿若无因飘落的轻雨」更新与补偿内容。

〓补偿内容〓
停服更新补偿：原石*300（停服维护每小时补偿60原石）
问题修复补偿：原石*300（补偿条目请参考对应补偿邮件）

〓补偿范围〓
停服更新补偿：2023/08/16 06:00前冒险等阶达到5级及以上的旅行者。
领取时间截至4.0版本结束前。
问题修复补偿：2023/08/16 06:00前冒险等阶达到5级及以上的旅行者。
请旅行者于2023/08/19 06:00前上线收取邮件。
制作组将在维护完成后5小时内，以游戏内邮件的形式，向旅行者发放相应补偿。邮件有效期为30天，请及时领取附件。

〓更新时间〓
2023/08/16 06:00开始，预计5个小时完成。

〓客户端更新方式〓
PC：关闭当前游戏后，重新打开启动器，点击更新；
iOS：进入App Store，点击更新；
安卓：进入游戏后，按照游戏内弹窗提示完成更新；
PS5™或PS4™：打开系统界面-选择原神-按下手柄OPTIONS键-选择确认升级；
若遇到任何安装问题，请联系客服，我们将为旅行者解决问题。

	处理器：第九代 Intel® Core i5™-9400F或同等配置；
	独立显卡：NVIDIA® GeForce® GTX 1050 Ti 或同等配置；

	内存：16GB 及以上配置；
20. 「七圣召唤」玩法更新：
	新增角色牌「七七」「丽莎」「阿贝多」及对应天赋牌；
	新增牌手名录中的邀约对局关卡和来客挑战中访客；
	新增行动牌「王下近侍」「黄金屋」「塞塔蕾」「藏锋何处」「愉舞欢游」；
	新增“观看对局”功能：
	（1）当好友在单人游戏状态进行「七圣召唤」牌局时，可以在“好友”界面看到其正在进行七圣召唤对局，并可在该界面以好友的视角观看其正在进行的对局（若好友处于多人游戏状态，则需要加入好友的世界）；
	（2）多人游戏队友进行「七圣召唤」牌局时，可以在“多人游戏”界面看到其正在进行七圣召唤对局，并可在该界面以队友的视角观看其正在进行的对局；
	（3）在多人游戏状态下，且另外两位队友之间进行「七圣召唤」队友邀请牌局时，可以通过裁判视角观看对局。在裁判视角下，可以同时看到双方的手牌和元素骰；
	（4）在「七圣召唤」牌局内的“设置”中，可以分别设置“允许或禁止好友观看自己的对局”，以及“允许或禁止多人游戏队友观看自己的对局”；
	（5）进行「七圣召唤」牌局时，可以看到正在观看你的牌局的观众列表，你可以选择将特定的观众移出列表；
21. 深境螺旋
	深境螺旋9-12层奖励「深秘圣物匣·一等」、「深秘圣物匣·二等」中的圣遗物套装「水仙之梦」和「花海甘露之光」，将替换为「逐影猎人」与「黄金剧团」；

	第九层地脉异常调整为：
	• 角色触发蒸发反应后的5秒内，该角色通过蒸发反应造成的伤害提升提高50%。
	第十层地脉异常调整为：
	• 角色对处于水元素附着状态下的敌人造成的伤害提高50%。
	第十一层地脉异常调整为：
	• 队伍中所有角色获得60%火元素伤害加成。
	• 队伍中所有角色获得60%水元素伤害加成。

	更新了深境螺旋第九层~第十二层的敌人阵容。

	从4.0版本更新完成后的第一次深境期数更新开始，两期渊月祝福分别如下：

	第一期：
	灵醒之月
	角色的生命值恢复时，该角色的攻击力提升10%，此效果持续10秒，至多叠加3层，每层独立计时。角色的生命值降低时，该角色的元素精通提升40点，此效果持续10秒，至多叠加3层，每层独立计时。角色生命值恢复与降低触发的效果分别独立存在，独立计时。

	第二期：
	盘转之月
	角色的生命值恢复或降低时，该角色获得40%所有元素伤害与物理伤害加成，持续3秒。

	※以上深境螺旋的更新将于9月1日4点起开始生效（服务器时间）。
