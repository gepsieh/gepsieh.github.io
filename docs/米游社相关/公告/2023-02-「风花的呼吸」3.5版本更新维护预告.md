「风花的呼吸」3.5版本更新维护预告

==================

![](https://upload-bbs.miyoushe.com/upload/2023/02/24/75276539/3db6e7482adfd08246409c8cebfbaa38_645152542880420322.png)

亲爱的旅行者：
制作组将于近期进行版本更新维护，维护期间无法登录游戏。请旅行者留意游戏时间。
维护完成后游戏将更新为全新版本，建议旅行者在WIFI 环境下升级。

〓更新时间〓
2023/03/01 06:00开始，预计5个小时完成。

〓客户端更新方式〓
PC：关闭当前游戏后，重新打开启动器，点击更新；
iOS：进入App Store，点击更新；
安卓：进入游戏后，按照游戏内弹窗提示完成更新。
PS5™或PS4™：打开系统界面-选择原神-按下手柄OPTIONS键-选择确认升级。
若遇到任何安装问题，请联系客服，我们将为旅行者解决问题。

〓补偿内容〓
停服更新补偿：原石*300（停服维护每小时补偿60原石）

〓补偿范围〓
停服更新补偿：2023/03/01 06:00前冒险等阶达到5级及以上的旅行者。
领取时间截至3.5版本结束前。
制作组将在维护完成后5小时内，以游戏内邮件的形式，向旅行者发放相应补偿。邮件有效期为30天，请及时领取附件。
更多更新内容，包括问题修复、问题修复补偿等详见将于2023/03/01 07:00发布的版本更新说明。

〓调整及优化〓
● 系统
1. 圣遗物和武器强化界面现在可以通过长按并移动选择多个所要消耗的圣遗物和武器；
2. 摧毁界面现在可以通过长按并移动选择多个圣遗物或武器；
3. 移动端当特定玩法内交互操作按键不可用时，该按键范围内可进行触控操作以调整视角；
4. 优化了武器和圣遗物界面相关操作和交互，修复了部分问题：
手柄模式下，武器和圣遗物强化至突破等级上限后，或武器精炼至满阶后，现在可通过“返回”按钮返回上一级界面；
手柄模式下，武器强化和圣遗物强化界面内武器、圣遗物右侧属性现在支持使用右摇杆上下滑动；
手柄模式下，在圣遗物界面进入对比界面时，再按一次相同按键即可收起对比；
修复了特定情况下，圣遗物界面内的圣遗物名称有小部分会被异常遮挡的问题；
优化了在角色界面对武器或圣遗物进行操作时，武器列表或圣遗物列表的移动效果；

● 音频
1. 优化了部分室内场景的音效混响效果；
2. 优化了部分界面的音效表现；
3. 优化了部分角色韩语、英语和日语语音台词，修复了部分语音文本错误的问题；
4. 优化了部分任务的韩语、英语和日语语音台词；

● 七圣召唤
1. 调整了「七圣召唤」投掷骰子阶段的界面表现；
2. 优化了「七圣召唤」中，召唤物和阵营出战状态的可用次数刷新逻辑；优化后，可用次数会优先取最高值；
例：假设一个召唤物的默认初始可用次数为2，而当前场上已存在该召唤物且其当前可用次数为4；优化前，假如再次创建该召唤物，则其可用次数会被刷新为2；优化后，其可用次数会取最高值，即为4。

● 其它
1. 优化了个别场景的画面表现;
2. 优化了「旅行者」命之座激活道具的来源描述；
3. 地图中「千壑沙地」地区的「愚妄行宫」区域，增加了一个指向洞口的道路（仅增加地图标识，实际地形未改变）；
4. 在「永恒绿洲」区域内不再能使用小道具「寻宝罗盘」；
5. 调整了成就“比勒琪丝的哀歌”在成就合集「须弥·饰金砂原·其之二」中的显示顺序；
6. 调整名片「成就·砂岚」名字为「须弥·砂岚」;
7. 调整了部分道具的分类名称：
敌人掉落的部分素材调整为“角色与武器培养素材”;
“天赋培养素材”调整为“角色天赋素材”
击败首领敌人后，消耗树脂获得的部分用于角色突破的素材由“角色培养素材”调整为“角色突破素材”；
由合成台合成的“小道具”调整为“消耗品”。
