〓本次更新内容〓
一、限时区域——「金苹果群岛」（该区域仅在2.8版本期间开放）
「金苹果群岛」
◇区域开启条件：
•冒险等阶≥32级
•完成魔神任务第二章·序幕「振袖秋风问红叶」
•完成莫娜传说任务映天之章·第一幕「在此世的星空之外」
◆活动期间，接取莫娜传说任务映天之章·第一幕「在此世的星空之外」所需的冒险等阶也会调整至32级
◆2.8版本结束后，将无法再前往「金苹果群岛」，也无法获取「金苹果群岛」上的宝箱、素材、任务奖励等，还请旅行者及时探索。
二、全新角色
「心朝乂安·鹿野院平藏(风)」（★4）
◇神之眼：风
◇武器：法器
◇鹿野院平藏的元素战技「勠心拳」长按可积蓄力量，以施展更为强大的拳势。蓄力期间，将持续获得能提升勠心拳威力的「变格」效果；松开技能或蓄力结束时，会向前打出勠心拳，造成风元素伤害。拥有4层「变格」时，将产生「正论」效果，使下一次勠心拳造成的伤害将进一步提高，并拥有更大的影响范围。
施放元素爆发「聚风蹴」，鹿野院平藏会腾跃至空中，向敌人踢出不动流·真空弹，在命中时炸裂，产生缉拿风穴，牵引附近的物体与敌人，造成风元素范围伤害。不动流·真空弹命中处于水元素/火元素/冰元素/雷元素附着下的敌人时，会为这些敌人施加「聚风真眼」效果，该效果会在片刻后爆发消失，并造成对应元素属性的范围伤害。不动流·真空弹至多对4名命中的敌人施加「聚风真眼」；敌人无法同时处于多种元素属性的「聚风真眼」状态下。
◇天领奉行的天才少年侦探，直觉敏锐，心思玲珑。
三、全新装备
「单手剑·『妖刀』」（★4）
普通攻击、重击或下落攻击命中敌人时，将卷起切落风，造成180%攻击力的范围伤害，并且使攻击力提升15%，持续8秒。该效果每8秒至多触发一次。
*参与2.8版本更新后的相关任务剧情，将能了解到更多关于该武器的信息。

活动期间，完成「逸速穿浪」与「幽象谜域」中指定任务，可获得原石、大英雄的经验、精锻用魔矿等奖励。完成「幽象谜域」挑战，可获得「花簇簇的漂流物」，累计获取「花簇簇的漂流物」达到指定数量可邀请「断罪皇女！！·菲谢尔(雷)」，使用「花簇簇的漂流物」可在活动商店中兑换智识之冕、角色培养素材、天赋培养素材等奖励。

从2.8版本更新完成后的第一次深境期数更新开始，三期渊月祝福分别如下：

第一期：
奋勇之月
角色登场后，每2秒获得一层「奋勇」效果，攻击力提升20%，该效果至多叠加3层。处于3层「奋勇」状态下的角色在重击命中敌人时，将在敌人的位置释放冲击波，造成真实伤害，每2秒至多通过这种方式释放一次冲击波。角色退场时，将清除所有的「奋勇」效果。

第二期：
风洗之月
角色对敌人触发扩散反应时，敌人的防御力降低12%，持续10秒。该效果至多叠加3层，每层持续时间独立计算。

第三期：
碎破之月
当前场上角色的普通攻击、重击或下落攻击命中敌人后，该角色普通攻击、重击与下落攻击造成的伤害提升12%，持续8秒。该效果每0.1秒至多触发一次，至多叠加4层，每层持续时间独立计算。角色退场后，将清除该效果。

*2.8版本更新的内容中，部分汉语配音效果可能会受录音条件影响。敬请各位旅行者谅解。
〓调整及优化〓
● 敌人
1. 下调了世界等级3-6的世界中，「野伏·阵刀番」、「野伏·机巧番」、「野伏·火付番」、「海乱鬼·炎威」、「海乱鬼·雷腾」、「飘浮灵」的攻击力；
2. 优化了「兽境猎犬」的受击表现；
3. 调整了「飘浮灵」的受击和被击飞表现；
4. 下调了「飘浮灵」的血量和抗打断能力，同时对应上调了飘浮灵「怒气」充满所需的生命值百分比；
5. 调整了「飘浮灵」的部分行为逻辑：在非「膨胀」状态下可以受到牵引效果影响；

● 系统
1. 拾取掉落物时，若其堆叠数量大于1，掉落物的图标也会显示其堆叠的数量；
2. 优化了圣遗物属性筛选，优先显示主属性为所筛选属性的的圣遗物；
3. 角色界面增加了「旅行者」的角色培养指引；

● 音频
1. “角色-资料”中增加了部分角色的赠礼语音；
2. 优化了部分角色攻击的音效效果；
3. 优化了部分过场动画的音量表现效果；
4. 优化部分角色，任务，NPC的日语、英语、韩语语音；
5. 调整了角色「风色诗人·温迪(风)」的一句韩语语音，调整后语音和字幕一致；

● 其它
1. 在同一期祈愿活动期间，再次进入祈愿界面时，会保留选中上次退出时所选择的祈愿界面；
2. 派蒙菜单界面的「指南」更换成「版本热点」；
3. 修复了手柄模式下，“角色选择”界面的按键冲突问题：在“角色选择”界面，好感奖励领取按键将隐藏，L3按键的对应功能为筛选；
4. 在手柄模式下，活动商城中不可购买或不可兑换的道具，其购买按钮及购买功能将会被屏蔽。
〓问题修复〓
● 任务
1. 修复了魔神任务「危途疑踪」中部分任务道具文本错误的问题；
2. 修复了在完成世界任务「扇底春秋」时，传说任务「匪石」的任务道具「老石」异常消失的问题；修复前遇到该问题的旅行者，消失的道具会自动补发到背包内。

● 敌人
1. 修复了BOSS「祸津御建鸣神命」召唤的「恶曜三钴」有小概率出现位置异常的问题；
2. 修复了角色在攻击BOSS「裂空的魔龙」的爪子时，有小概率出现锁定异常的问题；

● 角色
1. 修复了角色「护法夜叉·魈(风)」、「戎犬锵锵·五郎(岩)」在“角色-天赋”界面的动作存在穿模的问题，优化了「戎犬锵锵·五郎(岩)」、「风色诗人·温迪(风)」、「红叶逐荒波·枫原万叶(风)」在“角色-天赋”界面的服装飘带效果；
2. 修复了角色「『公子』·达达利亚(水)」施放元素战技进入近战状态，进行特定操作后元素战技冷却时间会出现异常的问题；
3. 修复了角色「兰生幽谷·夜兰(水)」瞄准射击时，部分角度下白纻飞练帔存在穿模的问题；

● 装备
1. 修复了角色「蔷薇魔女·丽莎(雷)」装备并激活圣遗物「来歆余响」四件套效果时，普通攻击有小概率额外触发一次物理伤害的问题；

● 音频
1. 修复了部分过场动画有概率出现音乐缺失的问题；
2. 修复了部分场景区域音效缺失的问题；

● 系统
1. 修复了手柄模式下，在武器替换界面能异常打开属性参考界面的问题；
2. 修复了在「尘歌壶」放置功能下雪隐鼬模型未正常显示的问题；
3. 修复了在角色界面，角色「旅行者·空」进行待机动作时表情缺失的问题；
4. 修复了在角色界面，角色「忍里之貉·早柚(风)」、「花坂豪快·荒泷一斗(岩)」、「猫尾特调·迪奥娜(冰)」、「智明无邪·烟绯(火)」在进行部分待机动作时，脚下倒影会异常闪烁的问题；
5. 修复了当邮箱中存在部分语言（如泰语）的邮件时，部分邮件珍藏盒邮件会无法正常展示的问题。
6. 修复了阅读教程后，有小概率无法正常获得原石奖励的问题；版本更新后，未正常发放奖励的教程条目将重置为有红点提示的状态，且阅读后会正常获得原石奖励；

● 其它
1. 修复了进入多人游戏模式前，主机进行特定操作，客机加入世界后，大世界敌人会异常维持在暂停游戏状态的问题；
2. 修复了部分武器特效在角色传送后有小概率消失的问题；
3. 修复了角色被触发碎冰反应后，切换角色或冲刺有概率异常重复播放碎冰特效的问题；
4. 修复了使用日语、德语、英语、印尼语、葡萄牙语、越南语进行游戏时，个别任务的过场动画存在部分字幕缺失的问题；
5. 修复了13个语言的部分文本错误，优化了文本（实际效果不变；旅行者可以通过 派蒙界面->设置->语言 切换游戏语言，到当前公告条目下查看对应语言的改动）。
