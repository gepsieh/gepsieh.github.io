米游社服务器升级公告（2019.11.12）

======================

![](https://upload-bbs.miyoushe.com/upload/2019/11/15/74757616/8c66fbe4c4bb490fcf2806ebea667834_4823479750799083103.jpeg)

米游社的大家：
晚上好。为了给大家带来更好的体验，我们将于2019年11月18日开始对服务器进行升级，升级时间约为1周。
升级期间使用当前网址访问米游社时如果出现异常，各位可通过备用网址：https://bbs.mihayo.com正常访问网页版米游社。初次使用备用网址浏览时需要重新登录。
**APP端将不受影响哦~**
 
备用网址指路：
米游社主页面：[https://bbs.mihayo.com/](https://bbs.mihayo.com/)

如升级期间无法正常访问网页版米游社，或有其他疑问，请通过[#米游社建议&反馈#](https://bbs.mihayo.com/dby/topicDetail/150)话题反馈，或加入米游社BUG QQ反馈群（879387354），我们将及时为您解答
感谢大家一直以来对米游社的支持，给大家带来不便非常抱歉。
