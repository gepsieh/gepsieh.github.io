关于1.4版本更新部分问题的说明小贴士

===================

![](https://upload-bbs.miyoushe.com/upload/2021/03/19/75276557/8eb236e485777f84d27192a5541fe42a_2879254698126815730.jpg)

亲爱的旅行者，你们好。
在本次1.4版本更新中，除了新内容的更新，我们也对旅行者反馈的许多问题进行了优化和修复。
在这里，开发组想针对其中部分问题的修复效果或调整初衷进行说明。
 
**1.修复了处于坐下状态的角色有概率可以被阳华生成的晶台托起的问题；**
**2.修复了多人游戏中，多个旅行者可以同时坐在同一个座位的问题；**
坐下是一个偏休闲体验的功能，可以让旅行者们在提瓦特冒险时，在游戏中的长凳、靠椅上，以及野外的休息点小憩。
在多人模式下，这些问题可能在无意间造成一些有趣的现象，但这些现象在部分情况下，可能破坏其他旅行者的体验，导致一些恶性问题。因此，开发组在经过评估后，决定在1.4版本中对这些问题进行修复。很抱歉我们对这些偏单机表现向的设计没有做好充分的稳定性评估，造成了如上问题。开发组会在今后的版本中更加关注这些功能设计的表现，并对可能导致恶性问题的隐患进行修复。
 
**3.修复了角色使用下落攻击时有概率被岩创造物卡住无法进行其他操作的问题；**
**4.修复了角色在跳跃过程中被怪物击中，有小概率被异常击飞较远距离的问题;**
**5.修复了部分角色在攻击且未击中目标后，在进入待机状态的瞬间跳跃，可进行下落攻击的问题；**
原神的物理系统是核心玩法中重要的一环，也是较容易出现问题的模块。物理系统相关问题有些严重破坏了旅行者的游玩体验，有些则影响了游戏玩法本身的平衡性。我们一直在持续努力完善游戏的物理系统，同时密切关注可能出现的各种问题以及现有优化或修复方案给旅行者体验带来的实际影响，希望给旅行者们带来更好的游戏体验。
 
**6.修复了角色「白垩之子· 阿贝多(岩)」命之座第6层「无垢之土」有概率未正常生效的问题；**
**7.修复了角色「白垩之子· 阿贝多(岩)」施放元素战技时存在短暂卡顿的问题。（3月18日新增修复）**
阿贝多的元素战技，依赖许多为它定制化开发的功能，另一方面，为了遵循开放世界的其他行为规则，我们也为他的元素战技添加了一些限制。
从1.2测试版本以来，开发组一直在持续尝试优化阿贝多的使用体验，但也因此可能带来了额外的问题。非常抱歉这些优化措施给旅行者们的线上体验造成了困扰。
 
除了上面提到的调整，在1.4版本更新后，我们还发现了以下问题：
**8.由角色（包括普通攻击、重击、下落攻击、元素战技、元素爆发伤害等）引发的剧变反应（包括碎冰、感电、超导、超载、扩散）伤害，将按照触发时对应角色自身的元素精通和相应元素反应加成值，进行动态实时计算。**
 
放到实例里来看的话，更新前后对比的情况如下：
更新前：【在1.0~1.3版本中】
砂糖触发天赋「触媒置换术」，为同队的甘雨增加元素精通 → 阿贝多触发天赋「瓶中人的天慧」后，为同队的甘雨再增加元素精通→ 甘雨施放元素爆发 → 元素爆发触发的超导反应伤害全程享受两次提升过的元素精通加成。
 
甘雨施放元素爆发 →  砂糖触发天赋「触媒置换术」，为同队的甘雨增加元素精通 →   阿贝多触发天赋「瓶中人的天慧」后，为同队的甘雨再增加元素精通→元素爆发触发的超导反应伤害无法享受到两次提升过的元素精通加成。
 
更新后 ：【1.4版本】
若甘雨受到了两个角色带来的元素精通加成，只要在该加成效果持续期间（无论是先受到加成再施放元素爆发还是先施放元素爆发再受到加成），元素爆发触发的超导反应均能享受到甘雨被提升过的元素精通加成。但该加成受到天赋持续时间约束，仅在两个天赋同时生效期间可达到峰值。
 
此现象还可能导致以下变化：
● 临时精通加成效果的有效覆盖率提升：砂糖天赋「触媒置换术」、阿贝多天赋「瓶中人的天慧」、教官套装的触发效果、武器「终末嗟叹之诗」的被动触发效果等；
● 不再能通过严格的技能施放手法，让某些技能全程享受峰值精通的叠加效果。（即旅行者常提到的“快照”说法）
经过讨论评估后，开发组认为1.4版本更新导致的上述问题并不影响元素精通相关效果的实际强度和极端情况下元素精通可堆叠的效果峰值，同时也使得相关效果变得的更容易触发和操作。因此，开发组对这一改变暂不另作调整，并将持续关注旅行者的线上体验和反馈。
 
感谢旅行者们长久以来的不断反馈，开发组也期望旅行者给予我们更多建议与反馈，我们会尽最大努力保障旅行者的游戏体验。
