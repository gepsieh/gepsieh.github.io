V2.0版本游戏问题集中反馈-8月19日更新

======================

新版本「不动鸣神，泡影断灭」现已开启！旅行者如果遇到问题想反馈，可以反馈到集中帖内，客服娘便可及时收集旅行者们的反馈。还请旅行者们按照以下格式将遇到的问题进行反馈，当日值班版主们会尽力及时为旅行者们做出解答！
 
〓已知问题〓
1.**（7月23日新增）**世界任务「执望三千里」有小概率出现调查点消失，无法被完成的问题。遇到该问题的旅行者，可以在服务器时间04:00（每日委托刷新后），通过派蒙菜单退出游戏重新登录，再次尝试交互。该问题将在2.1版本中修复。

2. 怪物深渊法师、大型冰史莱姆、大型岩史莱姆施放的首个护盾存在伤害吸收量异常的问题，该问题将在2.1版本中进行修复。
 
3. 圣遗物「追忆之注连」的4件套效果存在描述不明确的问题（实际效果未出现异常），现描述为：“施放元素战技时，如果角色的元素能量高于或等于15点，则会流失15点元素能量，使接下来的10秒内，普通攻击、重击、下落攻击造成的伤害提高50%”；
后续将调整为：“施放元素战技时，如果角色的元素能量高于或等于15点，则会流失15点元素能量，使接下来的10秒内，普通攻击、重击、下落攻击造成的伤害提高50%，持续期间内该效果不会再次触发”。该问题将在2.1版本中修复。
 
4. 角色「旅行者(雷)」的命之座第6层描述存在错误（实际效果未出现异常），
现描述为：“雷轰电转每引发2次威光落雷，就使下一次威光落雷造成的伤害提高100%，并为当前角色额外恢复1点元素能量”；
后续将更正为：“雷轰电转每引发2次威光落雷，则大幅增强下一次威光落雷，造成原本200%的伤害，并为当前角色额外恢复1点元素能量”。该问题将在2.1版本中修复。
 
5. 角色「猫尾特调·迪奥娜(冰)」施放元素战技「猫爪冻冻」形成护盾后，在持续时间内再次施放，会出现护盾基础吸收量未重置，仅持续时间被延长的问题。该问题将在2.1版本中进行修复。
