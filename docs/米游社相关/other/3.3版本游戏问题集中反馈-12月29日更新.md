〓已知问题〓
1. **（12月29日）**目前，角色「久世浮倾·流浪者(风)」在部分不允许使用技能的状态下，存在仍可使用元素战技的问题。该问题将在后续版本中修复，请留意后续公告。

2. **（12月16日）**目前，角色「花坂豪快·荒泷一斗(岩)」处于「怒目鬼王」状态时再次施放元素爆发，有小概率出现元素爆发无法正常生效的问题。该问题将在后续版本中修复，请留意后续公告。

3. **（12月12日）**目前，壶灵阿圆处可以异常购买到3.3版本纪行专属摆设图纸「『卓识之地的柔光』」。我们将在3.4版本，为3.3版本「灵津纪行」活动时间内（3.3版本更新后 ~ 2023/01/16 03:59），解锁了「珍珠之歌」的旅行者补偿全新纪行专属摆设图纸：「奥摩斯港三型通用起重机」；摆设的具体图标和摆放效果可以参考以下配图。非常抱歉给各位旅行者造成了不便。
![image](https://upload-bbs.miyoushe.com/upload/2022/12/12/75276550/d01f038af5a4ff352c535817d0873d9c_7006900520251175398.png)补充说明：已在壶灵阿圆处购买摆设图纸「『卓识之地的柔光』」并使用的旅行者，在解锁「珍珠之歌」后不会再获得摆设图纸「『卓识之地的柔光』」。

4. **（12月9日）**目前，使用部分华为设备游玩「七圣召唤」时，游戏客户端有概率出现闪退问题。遇到该问题的旅行者，可以在设备的 设置-关于手机-参数版本 界面点击右上角齿轮状按钮，将“参数自动下载并更新”打开；
当 设置-关于手机-参数版本 界面显示“设备参数优化 GT.GENC.1.17.22.101”即说明更新成功。
旅行者通过该更新即可解决闪退问题。若仍有问题，可以通过游戏内派蒙界面-反馈中提交反馈。
