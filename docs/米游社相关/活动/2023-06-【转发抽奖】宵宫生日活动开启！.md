【转发抽奖】宵宫生日活动开启！

===============

![](https://upload-bbs.miyoushe.com/upload/2023/06/20/76387920/b77f93ddd6a76e961321353d6d0d4bbf_1532570432184276451.png)

姆，旅行者们好呀~
今天酱酱在稻妻城外晒太阳，看到几个小孩子拿着糖果跑来跑去，是在玩什么游戏吗？
他们看起来很开心姆！酱酱等会也要找丘丘人和咕咕一起玩(›´ω`‹ )
6月21日00：00-23：59期间，旅行者可以进入「留影叙佳期」页面，为宵宫庆祝生日并收集相应的「画片」。
[>>点击进入展示页<<](https://webstatic.mihoyo.com/ys/event/e20220303-birthday/index.html?game_biz=hk4e_cn&bbs_presentation_style=fullscreen&bbs_auth_required=true&bbs_landscape=true&activity_id=20220301153521&utm_source=bbs&utm_medium=mys&utm_campaign=arti)
（6月22日0：00之后，旅行者仍然可从米游社APP-旅行工具中进入展示页，但将无法收集画片，请注意时间）

![](https://upload-bbs.miyoushe.com/upload/2023/06/20/76387920/8a5ed19c9fe5c8a98a258407a93f9fc8_3741829593368441344.png)

2023年6月21日-2023年6月23日18:00
开奖时间：活动结束后系统会结算中奖结果，并自动转发原帖@前三名中奖的旅行者

![](https://upload-bbs.miyoushe.com/upload/2023/06/20/76387920/7e332fe3e12cc1947d2f2ea53cfca788_6679054441279581340.png)

活动期间内，**转发本帖到自己的动态**，即视为参与成功。
 *旅行者们可以点击主帖底部的**【转发】**按钮进行动态转发；

![](https://upload-bbs.miyoushe.com/upload/2023/06/20/76387920/660bbde11accd608643d9c762fe8da77_8548044265845460881.png)

*目前动态功能仅支持米游社手机App，使用米游社网页端的旅行者们可以进入米游社App参与本活动~

**P.S 今天是宵宫的生日，不要忘记为她送上祝福哦٩(๑>◡<๑)۶~**

![](https://upload-bbs.miyoushe.com/upload/2023/06/20/76387920/427be91271611257e42d615133cd1399_1875368263713767058.png)

宵宫命定之日系列套装*20

![](https://upload-bbs.miyoushe.com/upload/2023/06/20/76387920/1152ad6e94661ef18dd513ea55c56926_7585958218259845730.png)

酱酱将从成功参与活动的旅行者中随机抽选20名赠送宵宫命定之日系列套装~

![](https://upload-bbs.miyoushe.com/upload/2023/06/20/76387920/5588e2502560fb7de2836a08463b080f_8792595659783505373.png)

1. 到达开奖时间后，系统将会结算中奖结果，并自动转发原帖@前三名中奖的旅行者（超出3名时，后续中奖的旅行者将会被省略）。
2.  收到中奖的系统通知后，旅行者可以点击该通知，直接前往中奖详情页，在详情页中点击【点击提交收货地址】按钮，填写收货地址即可。
