〓补偿内容〓
停服更新补偿：原石*300（停服维护每小时补偿60原石）
其余问题修复补偿：原石*300（补偿条目请参考对应补偿邮件）

〓补偿范围〓
停服更新补偿：2023/12/20 06:00前冒险等阶达到5级及以上的旅行者。
领取时间截至4.3版本结束前。
问题修复补偿：2023/12/20 06:00前冒险等阶达到5级及以上的旅行者。
请旅行者于2023/12/23 06:00前上线收取邮件。
制作组将在维护完成后5小时内，以游戏内邮件的形式，向旅行者发放相应补偿。邮件有效期为30天，请及时领取附件。

〓更新时间〓
2023/12/20 06:00开始，预计5个小时完成。

〓客户端更新方式〓
PC：关闭当前游戏后，重新打开启动器，点击更新；
iOS：进入App Store，点击更新；
安卓：进入游戏后，按照游戏内弹窗提示完成更新；
PS5™或PS4™：打开系统界面-选择原神-按下手柄OPTIONS键-选择确认升级；
若遇到任何安装问题，请联系客服，我们将为旅行者解决问题。

◆召唤华丽的「金花礼炮」，猛烈炮击前方的敌人，造成岩元素范围伤害。并在接下来一段时间内持续进行「支援炮击」，造成岩元素伤害。炮击命中敌人时，将为娜维娅积攒「裂晶弹片」。

第一期：
壮固之月
角色获取结晶反应产生的晶片后，将产生「壮固之能」状态。处于该状态下的当前场上角色造成的伤害提升15%，该状态持续8秒，至多叠加3层，每层独立计时。

迸断之月
角色获取结晶反应产生的晶片时，将进行计数，计数达到3次时，将在角色的位置释放冲击波，对附近的敌人造成真实伤害，并移除计数。此效果每6秒至多触发一次。

〓调整及优化〓

<details>
<summary>● 养成相关系统
</summary>

● 养成相关系统
1. 角色培养、武器强化、圣遗物强化界面取消了进行养成操作时出现的弹窗，养成信息与原有的界面整合；
2. 角色培养材料在值得铭记的强敌和首领敌人的奖励预览中，展示位置将更靠前；
3. 单次武器强化的可消耗素材数量由20提升至40，单次武器精炼可以选中多个精炼材料；
4. 调整了武器精炼的摩拉消耗规则：使用精炼等级不为1的武器为素材精炼武器时，其消耗的摩拉会基于素材精炼等级换算（例如，原精炼等级为1的4星武器，无论使用精炼等级为任何等级的武器作为素材，更新前其消耗的摩拉均为1000；更新后所需摩拉会基于精炼材料换算）；
5. “角色/升级”界面加快了培养相关特效的速度，并能使用快捷放入，一键放入能将角色提升到当前能提升的最高等级的角色经验素材；


</details>

● 圣遗物

4.  调整了“背包-圣遗物”界面的筛选功能：迭代了交互逻辑，支持更多筛选条件，并且能够同时筛选多个套装；
5.  圣遗物界面的排序功能，现在支持3条属性的排序规则（原支持2条）；
6.  合成台的“还圣奥迹”界面，支持一键快捷放入多个圣遗物；
