〓调整优化与问题修复〓

**▌调整&优化**
**• 怪物**
优化了怪物幼岩龙蜥和骗骗花在水域附近战斗时的钻地目标点选取逻辑。
**• 系统**
1. 优化了已装备道具的显示：已装备的圣遗物和武器会显示对应的角色小头像；
2. 优化了冒险之证-讨伐分类页签：不再按照地区分类，将按照怪物类型进行分类；
3. 优化了背包容量的显示：取消了负重限制，改为背包栏位。背包栏位分为3部分，分别是武器2000个栏位，圣遗物1000个栏位和其他材料2000个栏位。同种材料道具仍会堆叠在同一栏位。当对应背包栏位已满后，将不可继续获得同类道具，需对栏位进行清理；
4. 调整了原粹树脂的上限：提升至160（原为120）。
**• 其它**
1. 调整了纪行本周任务的原粹树脂累计消耗要求：调整为累计消耗1200点原粹树脂（原为：累计消耗1600点原粹树脂）；
2. 修改了常驻祈愿「奔行世间」的封面（未修改祈愿内容及规则，仅修改祈愿封面）；
3. 调整了传说任务「映天之章」第一幕的开启条件，需完成传说任务「四叶草之章」第一幕后才可解锁；
4. 调整了部分成就的名称；
5. 调整了预约达成累积奖励和全网预约1000万达成奖励活动的发放范围：取消原有领取限制。
    调整后预约达成累积奖励领取条件：无领取条件限制。
    调整后全网预约1000万达成奖励领取条件：冒险等阶≥10级。
    符合领取条件的旅行者均可通过游戏内邮件领取奖励。（已获得的旅行者不会重复发放）
