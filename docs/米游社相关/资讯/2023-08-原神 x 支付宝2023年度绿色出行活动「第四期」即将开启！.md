原神 x 支付宝2023年度绿色出行活动「第四期」即将开启！

==============================

亲爱的旅行者，
**原神 x 支付宝2023年度绿色出行活动「第四期」即将开启！**
**活动周期：**2023/04/22 00:00 ~ 2023/08/31 23:59
**第四期活动时间：**2023/08/03 10:00 ~ 2023/08/31 23:59

**============**
**活动说明**
**● 前往活动页：**旅行者可通过支付宝APP，首页搜索「原神」点击进入活动。
**● 参与活动：**旅行者可在活动页点击「攒里程」，完成指定浏览、绿色出行等任务，获得绿色里程。除活动页面任务外，每次用支付宝出行也会产生额外绿色里程，旅行者需要前往里程商城进行收取。
**● 兑换奖励：**旅行者所获得的绿色里程可用于兑换原神主题奖品或兑换周边抽奖机会。本期活动宫格兑换的奖品包含提纳里典藏出行皮肤、原石*80（限量800万份）以及更多游戏道具；周边抽奖的奖品包含原神x晨光奇只好玩 森林奇旅系列周边（提纳里款）、缘定礼遇系列徽章、出行优惠券等。

同时，旅行者们在活动期间通过绿色出行产生的绿色能量已突破20000吨，里程大奖——**【尘歌壶家具「若荒漠化为林野」】**已通过游戏内邮箱发放，旅行者们记得及时领取哦！
