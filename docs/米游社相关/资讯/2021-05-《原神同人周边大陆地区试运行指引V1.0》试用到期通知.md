《原神同人周边大陆地区试运行指引V1.0》试用到期通知

===========================

![](https://upload-bbs.miyoushe.com/upload/2021/05/18/75276539/b0bc1043ae8a8f5d0413e1b56c7cad27_7194074609897932363.jpg)

亲爱的旅行者，

《原神同人周边大陆地区试运行指引V1.0》已于2021年5月17日 0:00试用到期，感谢大家在试运行期间的关注与支持。近日，我们将着手梳理《原神同人周边大陆地区运行指引V1.0》，正式版指引将在整理完成后通知大家。

在新版指引发布之前，我们将暂停原神同人周边授权申请（在此之前进行的申请将不受影响，正常进行审核），由此给大家带来的不便，恳请谅解。
