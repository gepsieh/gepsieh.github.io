4.0版本「七圣召唤」全新卡牌说明

=================

![](https://upload-bbs.miyoushe.com/upload/2023/08/11/75276539/46754eb0ad99dfb4e66ec71dfda7666b_688419994285366250.png)

旅行者们好呀，4.0版本更新后，「七圣召唤」玩法将新增角色牌「阿贝多」、「丽莎」、「七七」和对应天赋牌，以及五张新的行动牌。一起来看看吧~

4.0版本更新后，前往猫尾酒馆与角色进行邀约对局，达成指定挑战目标可获得角色牌及对应天赋牌。新行动牌可在卡牌商店中购买。
*开发组将持续关注旅行者们的游戏体验，并对卡牌平衡性进行优化调整。卡牌信息请旅行者们以游戏内正式内容为准。
