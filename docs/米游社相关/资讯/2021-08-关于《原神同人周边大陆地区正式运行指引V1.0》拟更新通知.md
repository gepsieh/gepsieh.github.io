关于《原神同人周边大陆地区正式运行指引V1.0》拟更新通知

=============================

亲爱的旅行者，
非常感谢大家一直以来对《原神同人周边大陆地区正式运行指引V1.0》（以下简称《指引》）的支持，更感谢一直以来旅行者的每一条建议与意见。
近日来我们陆续收到关于指引施行期间部分现象的问询与建议邮件，对此我们非常重视。
经内部讨论，我们决定于近期再次完善指引内容，以便营造一个更加健康和友善的同人周边创作环境。
我们将于2021年8月20日12:00暂时关闭原神同人周边授权申请通道（在此之前进行的申请将正常审核，不受影响），并于新版《指引》发布后，再次开放申请通道。由此给旅行者们带来的不便，恳请谅解。
最后，无论申请通道是否开放，我们都欢迎各位旅行者通过邮件ystongren@mihoyo.com向我们提出更多的建议与意见，来帮助我们成长~
