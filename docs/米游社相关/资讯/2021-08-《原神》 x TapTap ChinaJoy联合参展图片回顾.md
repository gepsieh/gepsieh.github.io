《原神》 x TapTap ChinaJoy联合参展图片回顾

==============================

亲爱的旅行者，本次原神 x TapTap 2021ChinaJoy活动即将落下帷幕。受疫情影响，本次CJ活动内容为各位旅行者带来了诸多不便，我们深表抱歉，再次感谢各位旅行者的理解与陪伴。让我们一起回顾这三天的精彩瞬间吧！
 
特别提醒：明日的展台活动将在中午11点提前结束。
