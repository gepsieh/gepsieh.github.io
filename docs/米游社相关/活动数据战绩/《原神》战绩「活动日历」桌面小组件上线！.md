原神战绩「活动日历」桌面小组件功能于**米游社V2.79版本**正式上线啦！旅行者可以更直观地看到自己目前活动完成进度及体力恢复等情况了~
接下来就让阿姬带大家看看详细介绍&安装教学吧！

**=如何进入【原神战绩】=**
旅行者进入米游社App后，可通过【旅行工具-原神战绩】或点击【我的-我的角色】进入。
移动端的旅行者也可以点击此链接直接进入战绩：
[>>点击进入原神战绩<<](https://webstatic.mihoyo.com/app/community-game-records/?game_id=2&mys_source=bbs&mys_medium=mys&mys_campaign=arti)
*pc端的旅行者需要下载米游社app后才可以访问~

**=更新内容=**
「活动日历」桌面小组件会展示旅行者当前体力恢复状况、洞天财瓮的储存情况以及每日委托奖励的领取情况。同时，将**重点展示游戏内当前版本活动的完成进度及对应奖励**。
![image](https://upload-bbs.miyoushe.com/upload/2024/12/18/75276550/f18ba695ddef004f43c92828adb11e76_6115565953604675477.png)
并且，旅行者也可选择心仪的角色作为小组件的壁纸装饰自己的手机桌面啦~
![image](https://upload-bbs.miyoushe.com/upload/2024/12/18/75276550/523a9eb8df259523cdd7e75eedd4ff4e_1113004099522872325.png)
**=安装教学=**
**一、添加说明**
**1、iOS设备**
长按屏幕空白处 >> 点击屏幕顶部"+" >> 搜索米游社小组件 >> 找到【原神活动日历】>>添加小组件

**2、安卓设备**
安卓设备机型较多，各个设备机型之间会有差异，阿姬给大家列了几个常用设备的添加方式作为参考，具体操作需要旅行者根据自己的设备进行调试~

<details>
<summary>点击查看
</summary>

点击查看
● 华为
双指向内手势，拉起桌面管理 >> 点击"窗口小工具"（或"添加"，不同设备的按钮名称可能有一定差异），在已有的桌面组件的列表中找到【原神活动日历】添加

● 小米
长按桌面，拉起桌面管理 >> 添加小组件 >> 右上角图标 >> 安卓小部件 >>点击 “添加”，在已有的桌面组件的列表中找到米游社【原神活动日历】添加

● vivo
双指向内手势，拉起桌面管理 >>点击”添加”，在已有的桌面组件的列表中找到米游社【原神活动日历】添加

</details>

**二、切换壁纸**
**1、iOS设备**
长按小组件>>点击编辑小组件>>选择壁纸>>进入列表，点击想要的壁纸>>切换成功
![image](https://upload-bbs.miyoushe.com/upload/2024/12/18/75276550/fdadc2d5824aad228fd43955da8f62a5_2630087492968873353.png)![image](https://upload-bbs.miyoushe.com/upload/2024/12/18/75276550/8798f3a93a0a0e7140302ae5137c97a5_8149611851722330434.png)
**2、安卓设备**
点击小组件左上角设置图标>>进入列表，点击想要的壁纸>>切换成功
![image](https://upload-bbs.miyoushe.com/upload/2024/12/18/75276550/77a37c9aa72f7ef45b87eba411a29508_3632217244974261598.png)*部分安卓设备更新后，可能会出现组件两边未贴紧屏幕的情况，属于正常现象~
如果旅行者在实际使用上仍然存在一定的展示问题，可以尝试通过调整手机布局来避免该问题（如使用OPPO的旅行者，可以尝试调整桌面布局为4x5、5x5）。

以上就是原神战绩「活动日历」桌面小组件功能介绍&安装说明，小伙伴们可以将米游社更新至V2.79版本后进行体验~
