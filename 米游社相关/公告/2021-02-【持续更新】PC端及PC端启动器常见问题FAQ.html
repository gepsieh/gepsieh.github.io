【持续更新】PC端及PC端启动器常见问题FAQ

=======================

★2024年3月27日更新★
亲爱的旅行者：

《原神》公测开启以来，项目组收到许多关于PC版下载、更新的反馈。我们对大家的提问做了整理和汇总，想在此为大家一一进行解答。

▌原神客服的联系方式
1. 米游社app-右下角“我的”-点击“联系客服”
2. 游戏内打开派蒙菜单，点击“反馈”

▌机型适配
PC推荐配置
操作系统：Windows 7 SP1 64-bit 或 Windows 8.1 64-bit 或 Windows 10 64-bit（暂不支持 Windows 10预览版dev）
处理器：Intel Core i7 或同等处理器
内存：16 GB RAM
显卡：NVIDIA GeForce GTX 1060 6GB 及更高配置
DirectX 版本：11
存储空间：请预留 120GB 可用空间

PC最低配置
操作系统：Windows 7 SP1 64-bit 或 Windows 8.1 64-bit 或 Windows 10 64-bit（暂不支持 Windows 10预览版dev）
处理器：Intel Core i5 或同等处理器
内存：8 GB RAM
显卡：NVIDIA® GeForce® GT 1030 及更高配置
DirectX 版本：11
存储空间：请预留 120 GB 可用空间

〓启动器相关问题〓
▌启动器运行与更新相关问题
Q1：打开启动器或启动游戏时，提示：此文件的版本与正在运行的Windows版本不兼容。
A1：请确认Windows系统是否满足游戏支持的适配要求（见文首推荐配置列表），若设备性能较低，可能会出现无法安装、无法启动、黑屏、卡顿等现象。
PS：Windows7操作系统需达到Windows 7 Service Pack 1版本（补丁包SP1）才可支持游戏。（默认系统查看路径：开始->控制面板->系统和安全->系统）

Q2：点击启动器无反应。
A2：
(1) 旅行者可在游戏目录中，尝试点击launcher.exe启动启动器。查看是否有dump文件在启动器安装目录生成。若有，请将dump文件与日志文件通过客服反馈给我们；若无，则请将日志文件（游戏安装目录\Genshin Impact\logs路径下的所有文件）通过客服反馈给我们。

(2)在启动器安装目录下，找到“vc_redist.x64.exe”双击进行安装，
或前往下方微软官网下载安装最新的VC_redist.x64.exe ，再重新运行游戏。
https://support.microsoft.com/zh-cn/help/2977003/the-latest-supported-visual-c-downloads

若完成上述操作后仍无法解决，请查看当前设备是否同时具备独立显卡和集成显卡，若是，请禁用集成显卡后重新运行启动器。若否，请联系客服进行咨询。

Q3：打开启动器后，启动器主界面展示范围异常。（比如启动器界面超出显示器区域显示不完整）
A3：右键原神桌面图标属性后选择【兼容性】-【更改高DPI设置】勾选【替代高DPI缩放行为】，下面选【应用程序】，应用后重启启动器，可恢复正常显示范围。

Q4：提示：启动器检测更新失败，是否重新链接
A4：请检查网络设置：包括代理，hosts文件等。（可检查代理是否拦截了原神的网络连接。）

Q5：更新启动器时，遇到提示文件替换失败

A5:
（1）请打开任务管理器，找到“原神”进程（海外四服为“Genshin Impact”），右键点击结束任务后点击重试按钮。

（2）请在重启电脑后重试。

▌启动器插件安装相关FAQ
在《原神》游戏的后续版本中，KB3033929将成为游戏运行的必要组件，推荐使用Win7系统的旅行者们尽快更新该补丁。从2022/02/08起，启动器会自动为Win7系统的旅行者安装相关组件。

Q：我的启动器出现提示“游戏运行所必要的插件出现安装失败（1058）”，请问如何解决？

A：
1. 同时按下按键“WIN“和“R”以打开“运行”，在“运行”的输入框内输入"services.msc"后点击确认以打开”服务“。
2. 在打开的窗口中，找到”windows update“，双击打开。
3. 在弹出的窗口中，把”启动类型“选成手动，然后点击”应用“，再点击”确定“，这样相关服务就启动了。
4. 退出《原神》启动器（需要彻底退出，而不是缩小到系统托盘）。
5. 重新打开《原神》启动器，启动器会为系统安装补丁。
6. 安装成功后，根据启动器弹窗提示重新启动计算机即可。

Q：我的启动器出现提示“游戏运行所必要的插件出现安装失败（-2145124297）”，请问如何解决？

A：
1. 首先前往https://www.microsoft.com/zh-CN/download/confirmation.aspx?id=46148下载Windows6.1-KB3033929-x64补丁。
2. 然后将文件Windows6.1-KB3033929-x64.msu的后缀名修改为Windows6.1-KB3033929-x64.cab，并解压到一个新的文件夹中，不要在下载的默认文件夹中解压；
3. 同时按下按键“WIN“和“R”以打开“运行”，在“运行”的输入框内输入cmd并点击确认，以打开命令提示符。
4. 在命令提示符中输入dism /online /add-package /packagepath:"X:\路径\文件名.cab"，完成后按回车键执行安装。  (例如：我将文件解压到D盘，update 文件夹中，则我应该输入：dism /online /add-package /packagepath:"D:\update\Windows6.1-KB3033929-x64.cab")

〓客户端相关问题〓

▌游戏客户端下载与更新相关问题
Q6：我先前测试的安装包未卸载，我可以直接安装《原神》技术性开放测试PC版吗？
A6：请在安装《原神》技术性开放测试PC版时，先卸载先前测试的版本。

Q7：更新游戏时提示：磁盘空间不足，建议清理空间后重试。
A7：请确认游戏安装磁盘剩余空间大小，若不足120G，建议清理剩余空间后重新打开启动器更新游戏。

Q8：下载过程中提示：游戏文件校验错误，点击重试，尝试解决该问题

或下载过程中遇到以6或7开头的错误码弹窗（6XXX/7XXX）
A8：
方法一：请旅行者检查设备的网络环境是否良好，我们建议旅行者使用宽带下载代替WIFI， 并关闭代理；旅行者也可以通过进行网络重置以尝试解决问题（网络重置默认查看路径：windows设置->网络和Internet->网络重置）

方法二：请旅行者尝试将游戏安装在其他磁盘后再重试。

Q9：游戏安装过程中，提示：游戏文件解压失败。
A9：
（1）请确认游戏安装磁盘剩余空间大小，若剩余空间不足，建议清理剩余空间后重新打开启动器更新游戏。
（2）若剩余空间足够，请确认是否有安装杀毒软件，并确认原神安装路径中的相关文件是否有被拦截，若有被拦截，请尝试取消拦截，或信任文件，或退出杀毒软件后再重新打开游戏。

Q10：下载游戏时，下载进度/速度为0，或卡在某一点无法继续下载。
A10：
（1）请检查网络设置：包括代理，hosts文件等。（可检查代理是否拦截了原神的网络连接。）
（2）可尝试关机重启或右键以管理员身份打开启动器。

Q11：游戏下载过程中，提示：硬盘空间不足。但实际预留空间充足。
A11：
方法一：旅行者可以尝试修改硬盘格式，将FAT32转为NTFS。（修改硬盘格式存在风险，建议您在专业人员指导下，或者在保证了解更多专业信息下在进行操作）
方法二：旅行者可以尝试将游戏安装在其他磁盘后再重试。

Q12：游戏下载过程中，提示：
网络返回错误。
游戏文件下载错误，建议检查网络设置后重试
建议检查网络设置后重试

A12：请旅行者检查设备的网络环境是否良好，我们建议旅行者使用宽带下载代替WIFI， 并关闭代理；旅行者也可以通过进行网络重置以尝试解决问题（网络重置默认查看路径：windows设置->网络和Internet->网络重置）

Q13：下载过程中提示：程序无法响应。或是下载过程中遇到以8开头的错误码弹窗（8XXX）
A13：请尝试重新启动电脑或重新下载启动器。

Q14：下载安装游戏过程中遇到杀毒软件拦截，或提示：解压失败。
A14：下载安装打开游戏过程中，若出现杀毒软件拦截提示，请点击允许或信任操作，或退出杀毒软件。（若遇到360拦截，请重启电脑后再打开游戏）

Q15：游戏下载完成后，开始游戏按钮为灰色。
A15：请检查网络设置：包括代理，hosts文件等。（可检查代理是否拦截了原神的网络连接。）

Q16：下载安装游戏过程中遇到“-3005”错误码弹窗。
A16：
方法一：旅行者可以在“我的电脑”中找到“本次磁盘(C:)”，右键点击选择“属性”，使用“工具”页签下的“检查”功能检查游戏安装硬盘以尝试解决；

方法二：旅行者可以在“我的电脑”中找到“本次磁盘(C:)”，右键点击选择“属性”，在“硬件”页签选中对应的磁盘驱动器，点击“属性”在打开新的窗口；在新的窗口的常规页签中点击“改变设置”，最后在打开的窗口中的“策略”一栏中勾选“启用设备上的写入缓存”功能。

游戏运行相关问题
Q17：启动游戏后，未出现原神logo，一直处于白屏状态。
A17：
（1）请查看启动器安装目录中是否存在“DXSETUP”文件夹，若有，请双击该文件夹内的“DXSETUP.exe”尝试手动安装后再重启游戏；若没有，请下载安装DirectX 11。DirectX 11下载地址：https://autopatchcn.yuanshen.com/client_app/plugins/DXSETUP.zip
（2）请在更新显卡驱动后再次尝试启动游戏。

Q18：点击开始游戏，游戏程序无响应，无法启动游戏。
A18：
（1）请确认是否有安装杀毒软件，并确认原神安装路径中的相关文件是否有被拦截，若有被拦截，请尝试取消拦截，或信任文件，或退出杀毒软件后再重新打开游戏。
（2）退出游戏后把temp目录（在开始菜单搜索栏输入%temp%或WIN+R呼出运行输入%temp%）里的mhyprot2.sys和mhyprotect.sys删掉，如果删不掉重启电脑后重试，删除文件后退出所有杀毒软件，尝试重进游戏。
（3）可在游戏目录中启动YuanShen.exe进行尝试。（默认文件路径：Genshin Impact->Genshin Impact Game->YuanShen.exe）

Q19：启动游戏时提示：资源文件下载失败或网络问题，请重试 Code：-9908。
A19：
（1）可尝试多次重启游戏，请点击启动器“开始游戏”按钮右侧的“游戏本地文件管理”按钮->“一键修复”尝试修复文件。
（2）若游戏修复无法解决当前问题，请旅行者卸载启动器并删除游戏安装包，重新安装原神。

Q20：运行原神时遇到如下三种提示：
1、应用程序无法正常启动(0xc000007b)。请单击“确定”关闭应用程序。
2、由于找不到MSVCP140.dll，无法继续执行代码。重新安装程序可能会解决此问题。
3、Plugins：Fail to load 'XXX/Plugins/MTBenchmark_Windows.dll' with error '0x7e'.
A20：若旅行者遇到以上三种提示，请在启动器安装目录下，找到“vc_redist.x64.exe”双击进行安装，
或者去下方微软官网下载安装最新的VC_redist.x64.exe ，再重新运行游戏。
https://support.microsoft.com/zh-cn/help/2977003/the-latest-supported-visual-c-downloads

Q21：运行游戏时，提示：Error at hooking API “NtProtectVirtualMemory”.
A21：当您遇到该提示时，可以尝试通过控制面板来修复该问题。(路径：设置->控制面板->程序->程序和功能->搜索“Microsoft Visual C++”->对每一项都点击右键，选择更改->Repair)

Q20：点击“点击进入”后一片空白，过一会儿游戏闪退。没有报错信息，重装也没有用。

A21：旅行者可以在退出游戏后把temp目录（路径：在开始菜单搜索栏中输入%temp%；或按下“WIN+R”呼出“运行”，输入%temp%）里以mihoyocrash_开头的文件，通过客服反馈给我们。
